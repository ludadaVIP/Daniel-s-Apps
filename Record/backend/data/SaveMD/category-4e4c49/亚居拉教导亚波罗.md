# 真正建立教会的，不是身份，而是对神话语的谦卑

读《使徒行传》18:24-28时，我越读越觉得这一段经文很有意思。

亚波罗第一次出场，路加给了他极高的评价。

他生在亚历山大，一个当时世界著名的文化和学术中心；他有学问，善于辩论；他最能讲解圣经；他心里火热，将耶稣的事详细讲论，并且按着自己所知道的，教训人也很准确。

如果放到今天，他很可能就是一位很有影响力的讲员，一开口便能吸引许多人。

然而，路加紧接着写了一句话：

> “只是他单晓得约翰的洗礼。”

这不是说亚波罗错了，也不是说他是假教师，而是说，他所认识的真理还停留在一个阶段。他知道约翰所预备的道路，却还不知道后来已经发生的事情：耶稣已经完成救赎、已经复活、已经升天，圣灵已经降临，教会已经建立。

他所传讲的是真实的，却还不够完整。

更有意思的是，神安排谁来帮助他呢？

不是彼得。

不是雅各。

不是保罗。

而是一对普通夫妻——百基拉和亚居拉。

他们没有显赫的身份，没有耀眼的职分。他们原本只是做帐棚的，因为哥老丢下令犹太人离开罗马，他们不得不离开家乡，来到哥林多，与保罗一起工作。他们更像是一对流离失所的平信徒，而不是舞台中央的人物。

可是，就是这样一对夫妻，“将神的道给他讲解更加详细”。

每次读到这里，我都会停下来想一想。

一位最能讲解圣经的人，竟然需要两个普通信徒来帮助；而两个看似普通的信徒，却能够把神的道讲解得更加完全。

神似乎故意安排了这样一个画面，告诉我们一个非常重要的原则：**属灵的深度，并不完全取决于人的身份，而取决于人是否忠于神的话。**

今天，我们很容易把“职位”和“属灵”画上等号。

总觉得牧师一定最懂圣经，长老一定比信徒更明白神的旨意，讲台上的一定比台下的人更有看见。

然而，《使徒行传》这一幕，却轻轻地打破了这种思维。

圣经从来没有说，职分越高，对真理的认识就一定越深。

神赐给教会各种不同的恩赐，也把不同的领受分给不同的人。有人擅长讲解，有人擅长牧养，有人擅长教导，有人擅长劝勉。真正丰富教会的，不是所有智慧都集中在一个人身上，而是整个基督的身体彼此供应、彼此成全。

当然，这并不是说可以轻看牧者的职分。

圣经同样教导我们尊重、顺服那些忠心牧养群羊的人，因为他们承担着教导和守望教会的重要责任。

但是，这段经文提醒我们，**任何一个职分，都不能代替一个人持续学习神话语的责任；任何一个身份，也不能限制神借着谁来教导我们。**

更让我感动的，其实还不是百基拉和亚居拉，而是亚波罗。

如果换作今天，一个已经颇有名气的讲员，被一对普通夫妻指出还有不足，会是什么反应？

也许有人会觉得："我讲了这么多年，还需要你来教我吗？"

可是亚波罗没有。

圣经没有记载他争辩，没有记载他维护自己的面子，也没有记载他因为身份而拒绝别人。

他愿意听。

愿意学习。

愿意被补足。

正因为如此，他后来成为神手中更加有能力的器皿。

真正属灵的人，不是已经知道很多的人，而是即使已经知道很多，仍然愿意继续学习的人。

还有一个细节，也特别值得今天的教会学习。

百基拉和亚居拉，并没有在会堂里公开纠正亚波罗。

圣经说，他们“接他来”，然后才将神的道讲解得更加详细。

他们既维护了真理，也顾全了弟兄的尊严。

他们不是为了证明自己更懂，也不是为了赢得一场辩论，而是真心希望亚波罗能够更加认识神。

这让我想到，今天教会里许多讨论，常常容易变成争输赢、争对错、争谁更有看见。

可是，《使徒行传》里的这些人，没有人在争自己。

他们共同关心的只有一件事：**怎样才能让神的道被讲解得更加详细、更加准确。**

也许，这才是真正成熟的教会。

不是一个人什么都懂，而是每一个人都愿意在神的话语面前保持谦卑；不是靠身份建立权威，而是靠真理彼此成全；不是因为自己站在讲台上，就停止学习；也不是因为自己坐在台下，就觉得没有什么可以贡献。

百基拉、亚居拉和亚波罗，让我们看到了一幅极其美丽的属灵图画。

有人愿意教。

有人愿意听。

有人愿意补足。

有人愿意成长。

没有人的面子比真理更重要，没有人的身份比神的话更高。

这样的教会，神的道就不断被讲解得更加详细；这样的同工关系，福音也必不断向前推进。

愿我们都像百基拉和亚居拉一样，有爱心、有真理，也有智慧；更愿我们都像亚波罗一样，即使已经很有恩赐、很有知识，仍然愿意谦卑地接受弟兄姐妹的提醒。

因为，一个人真正的成熟，不是别人不能教他，而是无论神借着谁说话，他都愿意聆听，并且顺服神的话。
