# 一、先建立一个总概念

法语里没有像德语、俄语那样特别复杂的格变化，但**代词会根据在句子里的功能变化**：

* **主语代词**：谁在做动作
* **直接宾语代词**：动作直接作用到谁/什么
* **间接宾语代词**：动作通过介词“à”间接作用到谁
* **重读代词**：强调、对比、介词后面常用
* **所有格**：表示“我的、你的、他的”
* **反身代词**：自己做自己的动作
* **指示代词、疑问代词、人称代词等**：A1 先掌握常用部分

---

# 二、人称代词：主格（主语代词）

这是最先要背熟的。

| 人称     | 法语             | 中文                    |
| ------ | -------------- | --------------------- |
| 第一人称单数 | je             | 我                     |
| 第二人称单数 | tu             | 你                     |
| 第三人称单数 | il / elle / on | 他 / 她 / 人们、大家、我们（口语中） |
| 第一人称复数 | nous           | 我们                    |
| 第二人称复数 | vous           | 你们 / 您                |
| 第三人称复数 | ils / elles    | 他们 / 她们               |

## 例句

* **Je** suis étudiant.
  我是学生。

* **Tu** parles français.
  你说法语。

* **Il** travaille.
  他工作。

* **Elle** aime le café.
  她喜欢咖啡。

* **On** mange ici.
  人们/我们在这里吃饭。
  在口语里，**on** 很常代替 **nous**。

* **Nous** habitons à Paris.
  我们住在巴黎。

* **Vous** êtes chinois ?
  你是中国人吗？/ 您是中国人吗？

* **Ils** sont contents.
  他们很高兴。

* **Elles** parlent espagnol.
  她们说西班牙语。

# 三、宾格：直接宾语代词（COD）

法语里的“宾格”在 A1 阶段，最重要的是**直接宾语代词**，也就是“这个动作直接落到谁/什么上”。

| 人称    | 法语      | 中文      |
| ----- | ------- | ------- |
| 我     | me / m’ | 我       |
| 你     | te / t’ | 你       |
| 他/它   | le / l’ | 他 / 它   |
| 她/它   | la / l’ | 她 / 它   |
| 我们    | nous    | 我们      |
| 你们/您  | vous    | 你们 / 您  |
| 他们/她们 | les     | 他们 / 她们 |

## 例句

* Il **me** voit.
  他看见我。

* Je **te** comprends.
  我理解你。

* Je **le** connais.
  我认识他。

* Je **la** vois.
  我看见她。

* Il **nous** écoute.
  他听我们说。

* Je **vous** attends.
  我在等你们/您。

* Je **les** invite.
  我邀请他们/她们。

## 放在动词前

法语的代词位置非常重要，通常放在动词前：

* Je **le vois**.
  我看见他。

* Tu **la connais** ?
  你认识她吗？

# 四、间接宾语代词（COI）

这个很重要，但一开始常和“宾格”混。
它一般对应动词后面有 **à** 的对象。

| 人称    | 法语      | 中文        |
| ----- | ------- | --------- |
| 我     | me / m’ | 给我 / 对我   |
| 你     | te / t’ | 给你 / 对你   |
| 他/她   | lui     | 给他 / 给她   |
| 我们    | nous    | 给我们       |
| 你们/您  | vous    | 给你们 / 给您  |
| 他们/她们 | leur    | 给他们 / 给她们 |

## 例句

* Je **lui** parle.
  我跟他/她说话。

* Il **me** téléphone.
  他给我打电话。

* Nous **leur** écrivons.
  我们给他们写信。

## 怎么区分 COD 和 COI

* **Je vois Marie.** → 我看见玛丽。
  这里“Marie”是直接宾语，所以用 **la**：
  **Je la vois.**

* **Je parle à Marie.** → 我跟玛丽说话。
  这里有 **à**，所以是间接宾语：
  **Je lui parle.**

# 五、所有格：形容词式所有格（相当于“我的、你的、他的”）

A1 先学这个就够用了。
它的形式要跟**被修饰的名词**的性、数一致，不跟“人”一致。

| 人称       | 阳性单数  | 阴性单数  | 复数    |
| -------- | ----- | ----- | ----- |
| 我的       | mon   | ma    | mes   |
| 你的       | ton   | ta    | tes   |
| 他的/她的/它的 | son   | sa    | ses   |
| 我们的      | notre | notre | nos   |
| 你们的/您的   | votre | votre | vos   |
| 他们的/她们的  | leur  | leur  | leurs |

## 例句

* **mon** livre
  我的书

* **ma** maison
  我的房子

* **mes** amis
  我的朋友们

### 重要规则

如果后面的名词是**阴性单数**，但以元音或哑音 h 开头，不能用 ma/ta/sa，要用 mon/ton/son。

* **mon amie**
  我的女朋友 / 我的女性朋友

* **ton école**
  你的学校

* **son hôpital**
  他的/她的医院

这是 A1 很容易错的地方。

# 六、所有格：名词式所有格代词（“我的那个、你的那个”）

这个是另一种“所有格”，不是“mon/ma/mes”那种，而是代替名词。

| 人称      | 法语                                            | 中文        |
| ------- | --------------------------------------------- | --------- |
| 我的      | le mien / la mienne / les miens / les miennes | 我的那个      |
| 你的      | le tien / la tienne / les tiens / les tiennes | 你的那个      |
| 他的/她的   | le sien / la sienne / les siens / les siennes | 他的/她的那个   |
| 我们的     | le nôtre / la nôtre / les nôtres              | 我们的那个     |
| 你们的/您的  | le vôtre / la vôtre / les vôtres              | 你们的/您的那个  |
| 他们的/她们的 | le leur / la leur / les leurs                 | 他们的/她们的那个 |

## 例句

* Ce livre est **le mien**.
  这本书是我的。

* Ta voiture est rouge, la **mienne** est noire.
  你的车是红色的，我的是黑色的。

A1 不需要立刻掌握得很深，但你至少要知道它和 **mon/ma/mes** 不是一回事。

# 七、反身代词：自己做自己的动作

这类在法语里非常常见，尤其是“洗澡、起床、穿衣、休息”这些日常动作。

| 人称    | 法语      |
| ----- | ------- |
| 我     | me / m’ |
| 你     | te / t’ |
| 他/她   | se / s’ |
| 我们    | nous    |
| 你们/您  | vous    |
| 他们/她们 | se / s’ |

## 例句

* Je **me** lève.
  我起床。

* Tu **te** laves.
  你洗澡。

* Il **se** couche.
  他上床睡觉。

* Nous **nous** amusons.
  我们玩得开心。

* Vous **vous** reposez.
  你们/您休息。

* Elles **se** préparent.
  她们准备。

# 八、重读代词 / 强调代词

这个在法语里很重要，因为它常用于：

* 强调
* 比较
* 介词后
* 省略动词的回答

| 人称   | 法语    |
| ---- | ----- |
| 我    | moi   |
| 你    | toi   |
| 他    | lui   |
| 她    | elle  |
| 我们   | nous  |
| 你们/您 | vous  |
| 他们   | eux   |
| 她们   | elles |

## 例句

* C’est **moi**.
  是我。

* Avec **toi**, tout va bien.
  和你在一起，一切都好。

* Pour **lui**, c’est facile.
  对他来说，这很容易。

* Sans **eux**, c’est difficile.
  没有他们，这很难。

* Moi, j’aime le thé.
  我嘛，我喜欢茶。

# 九、法语里最容易混淆的几个点

## 1）tu 和 vous

* **tu**：熟人、一个人
* **vous**：正式、尊敬、多人

## 2）on 和 nous

* **on**：字面是“人们”
* 口语里常常表示“我们”

例如：

* On va au cinéma.
  我们去电影院。

## 3）le / la / les 和 lui / leur

* **le / la / les** = 直接宾语代词
* **lui / leur** = 间接宾语代词

例如：

* Je **le** vois.
  我看见他。
* Je **lui** parle.
  我跟他说话。

## 4）mon/ma/mes 不是“我 + 的”随便乱用

它要跟后面的名词配合：

* mon livre
* ma maison
* mes livres

# 十、A1 还要一起掌握的相关基础

虽然你问的是人称、主格、宾格、所有格、代词，但 A1 通常还会连着这些一起学：

## 1）冠词

| 类型   | 法语                       |
| ---- | ------------------------ |
| 定冠词  | le / la / les            |
| 不定冠词 | un / une / des           |
| 部分冠词 | du / de la / de l’ / des |

例子：

* **le** livre = 这本书
* **un** livre = 一本书
* **du** pain = 一些面包

## 2）常见动词

A1 必须先熟：

* être = 是
* avoir = 有
* aller = 去
* faire = 做
* venir = 来
* prendre = 拿、坐、吃

## 3）基本句型

* Je suis…
* J’ai…
* Je vais…
* Je fais…
* Il y a…

# 十一、最实用的 A1 代词总表

你可以先把这个背下来：

## 主语代词

**je, tu, il, elle, on, nous, vous, ils, elles**

## 直接宾语代词

**me, te, le, la, nous, vous, les**

## 间接宾语代词

**me, te, lui, nous, vous, leur**

## 所有格形容词

**mon, ma, mes / ton, ta, tes / son, sa, ses / notre, nos / votre, vos / leur, leurs**

## 重读代词

**moi, toi, lui, elle, nous, vous, eux, elles**

---

# 十二、你现在最该怎么学

最有效的方法不是死背表，而是用句子反复练：

* **Je suis…**
* **Tu as…**
* **Il me voit.**
* **Je lui parle.**
* **C’est mon livre.**
* **Le livre est à moi.**
* **Je me lève.**

只要你能把这些句型反应成自然反射，A1 的核心就稳了。
