这是一个非常好的问题，而且也是理解《但以理书》的关键。

**如果《士师记》要画一张"属灵循环图"，那么《但以理书》则需要画一张"世界历史时间轴"。**

很多人读《但以理书》觉得很乱，就是因为脑子里没有一条完整的历史线。其实，只要把这条线理顺，旧约后半段、被掳时期、两约之间、新约都会串起来。

# 十、为什么《但以理书》特别强调这些帝国？

现在你再回头看第二章的大像，就非常清楚了：

| 金属 | 帝国   | 时间                   | 特点                      |
| -- | ---- | -------------------- | ----------------------- |
| 金头 | 巴比伦  | 626–539 BC           | 荣耀、富强，尼布甲尼撒时代达到巅峰       |
| 银胸 | 玛代波斯 | 539–331 BC           | 幅员辽阔、法律严谨，允许犹太人归回       |
| 铜腹 | 希腊   | 331–168 BC（亚历山大及继承者） | 希腊文化、语言迅速传播，为新约时代预备共同语言 |
| 铁腿 | 罗马   | 63 BC起统治犹太地区         | 军事强大、道路系统发达，为福音广传预备环境   |
| 石头 | 神的国  | 永远                   | 基督的国度，永不败坏              |

这里还有一个非常有意思的现象。

你读《列王纪》《历代志》，主角始终是**以色列**；但到了《但以理书》，镜头突然拉远了。神仿佛把但以理带到高处，让他俯瞰整个世界历史。以色列虽然只是地图上的一小块土地，却始终位于这些世界帝国交汇的中心。神不是只管理犹大，也不是被巴比伦打败了；恰恰相反，**巴比伦、波斯、希腊、罗马都只是神救赎计划中的工具**。

因此，《但以理书》并不是一本单纯讲预言的书，而是从被掳时代开始，告诉神的百姓：**无论世界霸主怎样更替，真正坐在宝座上的始终是神。**这也是为什么你会发现《但以理书》和《启示录》如此相似——两卷书都不是为了满足人对未来的好奇，而是为了在动荡的历史中坚定信徒对神国度的盼望。
