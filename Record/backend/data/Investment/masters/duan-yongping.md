## 一句话

段永平（1961 年生，江西人）**中国最被低估的伟大企业家 + 投资人** —— 1989 年小霸王做电子游戏机 → 1995 年创立**步步高**（学习机 + DVD + VCD）→ 2001 拆分 OPPO + vivo（全球出货量前五智能手机品牌）→ 2002 起在美国做价值投资 + 2006 投网易**赚 100x** + 长期持有苹果 / 茅台；他**2006 年与 Buffett 共进午餐**（$62 万）；他**40 年坚持"本分"哲学** + 中国版"少做事"价值投资 —— **理解段永平 = 理解中国企业家如何用 Buffett 哲学创造世界级公司 + 个人世界级财富**。

## 早年（1961-1989）

### 童年（江西）

- 1961 出生于江西南昌
- 父母都是知识分子
- 文革期间艰难成长

### 教育

- 浙江大学无线电系（本科）
- 中国人民大学计量经济学硕士
- 80 年代当兵 + 当工人

### 早期工作

- 80 年代末在广东中山一家电子厂
- 升任副厂长

## 小霸王（1989-1995）

### 创立 / 早期成功

1989 年 28 岁段永平加入**怡华集团**（中山）：
- 接管濒临破产的 **小霸王**
- 1991 年推出"小霸王学习机" + 后来"游戏机"
- 主打**家庭学习** + 游戏（双兼容）

### 营销天才

- 邀请**成龙** 代言（"望子成龙小霸王"）
- 中央台广告爆款
- 1995 年小霸王成中国家喻户晓品牌

### 离职原因

1995 年段永平**主动离开** 小霸王：
- 与怡华集团股东冲突
- 要求**股份激励** 被拒
- 决定自己开

## 步步高（1995-2001）

### 创立

1995 年段永平 + 团队**集体离开** 小霸王 → 创立 **步步高**：
- 在广东东莞
- 起始资金 $100 万
- 段永平拥有大量股权

### 三大产品线

1. **复读机**（学英语）
2. **VCD / DVD 播放机**（家庭娱乐）
3. **教育电子产品**（学习机）

### 营销策略

- 继续与**成龙** 合作
- 央视黄金时段广告
- 1996-2001 步步高成中国**最知名电子品牌**之一
- 年营收破 **$10 亿**

### 三家拆分（2001）

2001 年段永平**拆分** 步步高：
- 教育电子 → 步步高（保留品牌）
- VCD / DVD → 沃步（后来变化）
- 通信 → **OPPO + vivo**（陈明永 + 沈炜 各管一家）

**关键决策**：
- 段永平**不直接管理 OPPO + vivo**
- 但**控制股份** + 提供哲学指导
- 让职业经理人独立运营

## 移居美国 + 价值投资（2001-至今）

### 移居美国

- 2001 年带家人**移居美国 Palo Alto**
- 40 岁开始**全职投资**
- 中文论坛"**色诺芬尼"** 早期分享投资心得（后来雪球）

### 投资网易（2002-2003）

- 2002 年网易股价 **$0.64**（SEC 调查 + 接近退市）
- 段永平大量买入 → 200 万美元
- 2003 年开始反弹
- 2007 卖出 → **100x 回报** + 净赚 $2 亿

### 巴菲特午餐（2006）

2006 年段永平**以 $62 万** 拍得与 Buffett 午餐：
- 携朋友 + 黄峥（拼多多创始人，当时是段永平投资助手）
- 与 Buffett + Munger 4 小时交流
- 后续公开分享心得

### 长期持有苹果

- 2011-2012 开始买 Apple
- 2024 仍是大量持有
- 段永平**早 Buffett 5 年看好 Apple**

### 长期持有茅台

- 2013 年开始大量买入贵州茅台
- 持有至今
- 大幅获利

### OPPO + vivo 的崛起

- 2010 年代 OPPO + vivo 在中国 + 海外**起飞**
- 2024 年全球智能手机出货量分别 **前 5**
- 段永平作为最大股东**净资产爆发**

### 个人净资产

- 2024 估计 $100-200+ 亿
- 主要来自 OPPO + vivo + 投资组合
- **极度低调** —— 不在富豪榜常出现

## 投资哲学（5 大核心）

### 1. "本分"

段永平最核心哲学：

> "**做对的事，把对的事做对。**"

—— 做正确的事 + 把正确的事做正确。

"**本分**" = 真诚 + 道德 + 长期 + 客户至上。

OPPO / vivo / 步步高 **企业文化核心** 都是"本分"。

### 2. 不做"不懂"

> "**不熟不做。**"

段永平不投自己不懂的：
- 早期不投房地产
- 不投互联网平台（虽然投网易，但因为研究透）
- 不投加密
- 不投高频交易

### 3. 长期持有

- 网易持有 5+ 年
- Apple 持有 10+ 年
- 茅台持有 10+ 年
- **不频繁交易**

### 4. 用 Munger 的思考方式

段永平公开承认：

> "**Buffett 教我投资，Munger 教我思考。**"

- 跨学科 mental models
- 第一性原理
- 长期视角

### 5. 拒绝管理 = 长期 alpha

段永平不亲自管理 OPPO + vivo：
- 让职业经理人独立
- 自己专注**投资 + 哲学**
- 反共识："**不管理是最好管理**"

## 经典操作

### 1. 2002 网易 100x

2002 年网易股价被 SEC 调查打到 $0.64，段永平判断核心业务 + 游戏前景未变 → 集中买入 → 2007 卖出，100 倍回报，单笔净赚 $2 亿。这是中国价值投资史上最经典案例之一。

### 2. 2011 Apple

- 2011 开始大量买入 Apple
- 当时市场担心 Jobs 死后失去创新
- 段永平判断："**生态 + 品牌足够**"
- 持有 10+ 年 → 大胜

### 3. 2013 茅台

- 2012-2013 茅台 + 五粮液危机（反腐败 + 塑化剂）
- 段永平大量买入茅台
- 持有至今
- **5-10x** 回报

### 4. 长期不持腾讯

- 段永平**没买**腾讯（争议）
- 后来公开承认"看不懂"
- "**不熟不做**" 的应用

### 5. 拼多多投资（黄峥）

- 早期教过黄峥（拼多多创始人）
- 黄峥是段永平的早期助手
- 段永平间接投资拼多多
- 黄峥后期独立运营
