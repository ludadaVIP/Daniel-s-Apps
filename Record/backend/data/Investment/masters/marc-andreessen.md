## 一句话

Marc Andreessen（1971 年生）22 岁写 **Netscape 浏览器**（1994）—— 引爆**互联网商业化**；后来创立 **Andreessen Horowitz (a16z)** —— 投资 Facebook / Twitter / Airbnb / Stripe / Lyft / Coinbase / GitHub / Slack 等 + 2024 年管理 $450 亿；他 2011 年的著名宣言"**软件正在吞噬世界**"成为科技投资的核心论述；**Andreessen 代表现代 VC + 创始人友好 + 持续反共识** —— 既是工程师起家，也是哲学家思考。

## 早年（1971-1994）

### 童年

- 1971-07-09 出生于 Iowa 州 New Lisbon, Wisconsin（农场附近）
- 父母是种子销售员
- **9 岁** 自学 BASIC 编程
- 极度爱读书 + 极度好奇

### 教育

- University of Illinois Urbana-Champaign（计算机科学）
- 1992 年 21 岁 加入 **NCSA**（国家超级计算应用中心）

### Mosaic 浏览器（1993）

22 岁的 Andreessen 在 NCSA 写 **Mosaic** —— 第一个流行的图形界面浏览器：
- 1993 发布
- 几个月内成最流行浏览器
- **改变互联网从文本到图形**
- 引爆 web 商业化

## Netscape（1994-1999）

### 创立

1994 年 Andreessen + Jim Clark（Silicon Graphics 创始人）创立 **Netscape Communications**：
- Andreessen 23 岁
- 商业化 web 浏览器
- 6 个月内开发 **Netscape Navigator**

### 爆发性增长

- 1994-12 发布 Navigator
- 1995 年达到 **80%** web 浏览器市场份额
- 1995-08-09 **IPO**（$28，开盘 $71）—— **dotcom 时代的起点**
- Andreessen 24 岁 + $58M

### Browser Wars（1995-1999）

Microsoft 1995 推出 Internet Explorer：
- 与 Windows 捆绑（免费）
- 法律灰色地带
- Netscape 市场份额持续下滑
- 1998 AOL 收购 Netscape（$42 亿）—— Andreessen 净赚 $200M

### 教训

- 失去对 Microsoft 的注意（[[red-queen-effect]] 失败）
- 没建立**结构性护城河**
- **没收费模式** —— 给免费产品

## Loudcloud / Opsware（1999-2007）

### Loudcloud（1999）

Andreessen + Ben Horowitz 创立 **Loudcloud**：
- 早期 SaaS 模型
- 几乎倒闭 2001
- Pivot 到企业软件 → 改名 **Opsware**

### Opsware 卖给 HP（2007）

- 2007 年 HP $16 亿收购 Opsware
- 这次 Andreessen 学到企业软件 + 销售 + IPO 流程

## Andreessen Horowitz / a16z（2009-至今）

### 创立

2009 年 Andreessen + Ben Horowitz 创立 **a16z**：
- 首期基金 $3 亿
- 创新模式：**创始人友好** + **全方位支持**（PR / 招聘 / 销售 / 法律）
- 颠覆传统 VC 模式

### 业绩

| 时期 | a16z |
| --- | --- |
| 2009 首期基金 | 4 家投资贡献 80% 回报（Slack / Lyft / Coinbase / Skype） |
| 2024 AUM | **$450+ 亿** |
| 顶级被投 | Facebook / Twitter / Airbnb / Stripe / Lyft / Coinbase / GitHub / Slack |

### 创始人友好模式

a16z 的创新：
- 提供**专门团队**支持创业公司
  - PR 公关
  - 招聘
  - 销售关系
  - 监管
  - 法律
- 之前的 VC 只提供资金 + 一般建议
- **a16z 颠覆传统 VC 模式**

### 加密 vertical（2018-至今）

- 2018 a16z 设立专门**加密基金**
- 投资 Coinbase / Solana / Optimism / 等
- 2022 年 a16z crypto 第 4 期基金 **$45 亿**
- Andreessen 公开支持加密

## "软件正在吞噬世界"（2011）

2011 年 Andreessen 在 WSJ 发表**著名文章** "**Why Software Is Eating The World**"：

### 核心论点

> "**Software is eating the world.**"

—— 软件正在吞噬世界。

**预测**：所有行业都将**被软件公司颠覆**：
- 零售（Amazon）
- 媒体（Facebook / Twitter）
- 出租车（Uber）
- 酒店（Airbnb）
- 金融（Stripe / Square）

**2024 视角**：完全正确 —— 这篇文章预言了 2010s 科技革命。

## 投资哲学（5 大核心）

### 1. 创始人友好

- a16z 设立专门团队**支持创始人**
- 不只是钱
- 长期关系

### 2. 押大趋势

- 软件吞噬世界（2011）
- 移动互联网（2010s）
- 加密（2018-）
- AI（2022-）

### 3. [[power-law]] 集中

- 顶级投资贡献绝大部分回报
- 不分散
- 大额下注

### 4. 长期持有

- 投资后通常持有 10+ 年
- 不急着退出

### 5. 思想 leadership

- Andreessen 是**最活跃 VC 思想家**之一
- Twitter / blog / podcast
- 影响整个科技生态

## 10 个值得记住的名言

1. **"Software is eating the world."**
   软件正在吞噬世界。

2. **"The number one most important rule of investing is don't lose money."**
   投资第一规则：不要亏钱。

3. **"The two biggest mistakes you can make in startups: building too much, or hiring too many people."**
   创业最大两个错：**建太多 + 雇太多**。

4. **"Strong opinions, loosely held."**
   强观点，松握有。

5. **"It's easy to overestimate the impact of change in the short term and underestimate it in the long term."**
   短期容易高估变化，长期容易低估。

6. **"The world is a very malleable place. If you know what you want, and you go for it with maximum energy and drive and passion, the world will often reconfigure itself around you much more quickly and easily than you would think."**
   世界很容易改造。如果你知道想要什么 + 全力以赴，**世界会比你想的更快为你重新配置**。

7. **"The companies that are going to change the world are the ones that don't fit traditional patterns."**
   改变世界的公司**不符合传统模式**。

8. **"Don't fight Moore's Law."**
   不要和摩尔定律对抗。

9. **"The biggest risk is not taking any risk."**
   最大风险是**不冒任何风险**。

10. **"Markets are voting machines short term, weighing machines long term."** (引用 Graham)
    短期投票机，长期称重机。

## 反面 / 争议

### 1. Netscape 失败的教训未充分总结

- Netscape vs Microsoft 失败
- Andreessen 后期承认错
- 但**没有深入反思**

### 2. 加密押注争议

- a16z 大量投资加密
- 2022 LUNA / FTX 后争议
- 但 Andreessen **仍然**坚持

### 3. 公开政治立场

- 2024 公开支持 Trump
- 与硅谷部分人对立
- 但他坚持自己立场

### 4. "Techno-Optimist Manifesto"（2023）

- Andreessen 发表"科技乐观主义宣言"
- 极度乐观技术
- 部分人认为过度乐观
- 部分人共鸣

### 5. 部分投资表现一般

- a16z 部分投资表现一般
- 但顶级投资足够回报

## 给我们的启示（5 条）

3. **[[power-law]] 集中** —— 顶级投资足够。

4. **思想 leadership** —— Andreessen 公开写作影响生态。
