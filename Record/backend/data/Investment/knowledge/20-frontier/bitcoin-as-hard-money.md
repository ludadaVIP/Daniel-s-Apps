## 一句话

比特币没赢在技术——它赢在**用算法实现了凯恩斯之前的"硬货币"**：供应绝对刚性、不能被任何主体超发、抗没收；在央行无限 QE 的时代它成了"数字黄金"叙事的最佳载体，**13 年从 $0 到主流资产**。

## 53 年货币史的背景（理解 BTC 必须知道的）

### 1971 尼克松休克

1971-08-15 尼克松宣布美元与黄金脱钩——结束了布雷顿森林体系。

**这是 BTC 故事的真正起点**：从此世界进入**纯法币时代**，所有货币都由中央银行无限创造。

数据：
- 1971 年美元 M2 货币供应：**$0.69 万亿**
- 2024 年美元 M2：**$21.0 万亿**（53 年 **30 倍**）
- 同期美国 GDP 增长 21 倍——**货币供应增速持续超过经济增速**

### 法币恶性通胀的频率

很多人以为"法币崩溃"是少见事件。实际数据：

| 国家 | 时期 | 通胀峰值 |
| --- | --- | --- |
| 魏玛德国 | 1923 | 月度 +29,500% |
| 匈牙利 | 1946 | 日度 +207%（最快记录） |
| 津巴布韦 | 2008 | 月度 +7.9 万亿% |
| 委内瑞拉 | 2018 | 年度 +130,060% |
| 阿根廷 | 2023 | 年度 +211% |
| 土耳其 | 2024 | 年度 +65% |

**过去 100 年至少 50 个国家发生过 50% 年通胀以上**。法币失败不是边缘事件——是周期性常态。

### 现代央行扩表

| 年份 | 美联储资产负债表 | 全球央行总资产 |
| --- | --- | --- |
| 2008-01 | $0.9T | $9T |
| 2014-12 | $4.5T | $20T |
| 2020-03 | $4.2T | $24T |
| 2022-04 | **$9.0T** | $42T |
| 2024-12 | $7.4T | $37T |

**14 年扩张 8 倍**——比 BTC 出现前任何 14 年都激进。这是 BTC "数字黄金"叙事的宏观背景。

## BTC 的"硬"在哪里（4 个具体维度）

### 1. 总量绝对刚性

- 写在协议里：**2100 万枚**，不能改
- 改了就**不是 BTC**——这是网络共识的核心
- 黄金每年新增供应 1.5-2%；BTC 2024 减半后年发行率 **<1%**

### 2. Stock-to-Flow 比率（最硬的指标）

Stock = 已存在数量，Flow = 年新增数量。这是黄金行业评估"硬度"的传统指标。

| 资产 | Stock-to-Flow |
| --- | --- |
| 美元 (M2) | ~12 |
| 银 | ~22 |
| 黄金 | ~62 |
| **BTC (2020 减半后)** | **~56** |
| **BTC (2024 减半后)** | **~115** |
| BTC (2028 减半后) | ~230 |

**2024 年后 BTC 的 S2F 已超过黄金**。这是数学事实，不是观点。

### 3. 无发行方 = 无"政策"

- 黄金：无发行方，但开采可以加速
- 美元：发行方是 Fed，可以无限增发
- BTC：**无发行方，开采速度由协议固定**

意味着**没有任何主体能强制 BTC 通胀**——这是历史上独一无二的属性。

### 4. 抗没收

- **黄金**：1933 年罗斯福行政令 6102 强制美国人交出黄金
- **银行存款**：2013 塞浦路斯强制冻结
- **BTC**：只要私钥在你脑子里（12 个助记词），**没有政府可以没收**

这就是 [[2010-bitcoin-early-believers]] 里阿根廷人 Wences Casares 强力推荐 BTC 的根本原因——他经历过法币没收。

## 中本聪的密码学血缘

中本聪不是凭空出现。他站在 30 年 cypherpunk 运动的肩膀上：

| 年份 | 关键技术 | 贡献者 |
| --- | --- | --- |
| 1976 | 公钥密码学 | Diffie-Hellman |
| 1991 | PGP | Phil Zimmermann |
| 1997 | Hashcash (PoW 原型) | Adam Back |
| 1998 | B-money 方案 | Wei Dai |
| 1998 | Bit Gold 方案 | Nick Szabo |
| 2008 | Bitcoin | "Satoshi Nakamoto" |

**中本聪只是把已有的 5 个零件以正确方式组合**。他的创新是**完整系统**，不是单个突破。

中本聪身份至今未明。最可能的候选：
- **Hal Finney**（已故，2014）—— 第一个收到 BTC 转账的人
- **Nick Szabo** —— Bit Gold 设计者
- **Adam Back** —— Hashcash 发明者

可能是这几个人的合作。重要的是：**中本聪在 2010 年消失后 BTC 仍然运作 16 年**——证明这不是依赖个人的项目。

## 早期布局的两类人（[[2010-bitcoin-early-believers]] 深度分析过）

- **李笑来**：2011-2013 重仓买入约 $200K，平均成本 $1-3。布局逻辑：硬通货叙事 + 网络效应
- **孙宇晨**：更晚（2013-2017），不是 BTC 信仰者，是叙事 + 发币杠杆变现
- **Hal Finney**：技术信仰，2009 年挖矿
- **Wences Casares**：阿根廷恶性通胀经验 → 2011 年立刻理解 → 2013 年向 Bill Gates / Larry Summers 推荐

**4 类人路径不同，但都满足一个条件：早期把这件事当真 + 仓位重到改变人生**。

## 反面 / 边界（必须知道的）

### 1. 不是生产性资产

BTC 不产生现金流。DCF 估值完全不适用。**Munger 终其一生反对 BTC**——他认为没有现金流的东西是"投机"，不是"投资"。

这个反对**有理论价值**：BTC 的价值完全依赖**他人对它的看法**。如果有一天大家都觉得 BTC 没价值，它就归零。这种"反身性资产"不像 Apple 那样有底层利润支撑。

### 2. 能源消耗争议

PoW 挖矿每年耗电约 **150 TWh**（相当于波兰一国年用电量）。L2 / 闪电网络 / Lightning 是回应，但 BTC 主链本身不会变。

### 3. 政府监管 0-1 风险

- **中国 2017 + 2021**：两次全面禁止
- **印度**：反复政策摇摆，至今未定
- **美国**：ETF 批准 = 默认合法化，但 SEC 仍对其他加密项目敌对
- **萨尔瓦多 2021**：法定货币，IMF 反对，2024 部分撤回

**单一政府禁止不致命**（中国禁了 BTC 仍然涨），但**全球协调禁止**会是巨大风险。这个概率目前极低，但不为 0。

### 4. "数字黄金" vs "支付层"未盖棺论定

- **数字黄金叙事**（主流）：储值，类似黄金
- **支付层叙事**（少数）：日常使用
- **如果只能储值不能用**，长期需求受限——这是 BCH 阵营的核心担忧

### 5. 80% 回撤是常态

2014 (-83%)、2018 (-83%)、2022 (-77%)。**如果心理上扛不住**——仓位就该极小。

### 6. 早期信仰者的存活偏差

我们今天讲的"早期赢家"是因为他们坚持下来了。当时同样信仰但在 $1000 卖光、或者被 Mt. Gox / FTX 损失的人没故事留下。

**早期信仰者的真实生存率可能不到 20%**。

## 给投资者的启示

1. **理解货币史比理解 BTC 技术重要**。BTC 价值起点是法币失败的历史 + 央行无限 QE。如果你不信法币会失败，你不会信 BTC；如果你信，你必然押注。

2. **仓位决定结果，不是判断**。Wences 推荐"小到能输得起"的仓位——这是教科书答案。如果你只买 $100，即使涨 1000 倍只是 $10 万——不改变人生。1-5% 总资产是合理范围。

3. **自托管 = 财富防御**。MtGox / FTX 教训反复。把"crypto 投资"委托给中心化机构等于绕开 BTC 设计的核心承诺。

4. **变现纪律和持有纪律同样重要**。Hal Finney 浮盈未变现的案例（[[2010-bitcoin-early-believers]]）。周期性减 10-20% 仓位让你两边都拿。

5. **下一个 BTC 大概率不是 BTC**。第一次范式转换的红利已被定价。下一次范式（AI 算力 token? 能源 token? Privacy coin?）的形态会完全不同。
