## 一句话

Hamilton Helmer 在斯坦福商学院教 + 自己运营 Strategy Capital 30 年的研究：**只有 7 种结构能让公司持续赚取经济利润**（业内称 7 Powers）；这是 2016 年后取代 Porter 5 Forces 的现代护城河框架——**专为投资人设计**，而不是 Porter 那种为咨询师服务的通用工具。

## 为什么不再用 Porter 5 Forces

Michael Porter 1980 年提出 5 Forces（供应商议价、买家议价、新进入者威胁、替代品威胁、竞争对手强度）。**统治商学院 35 年**。

**但 Porter 5 Forces 对投资人有 3 个问题**：

### 1. 描述性，不可操作

5 Forces 告诉你"这个行业竞争激烈"——但不告诉你"哪家公司能在激烈竞争中赢"。

### 2. 静态视角

Porter 看的是"当前结构"。但投资是**对未来的下注**——需要识别"现在没有但未来会建立 power 的公司"。

### 3. 不区分 Power 类型的强弱

5 Forces 把所有"竞争优势"放一起。**但护城河有等级**：网络效应 >> 规模 >> 品牌 >> 转换成本。Porter 没区分。

Helmer 2016 出版 7 Powers，**专门为投资人解决这 3 个问题**。

## Helmer 对 Power 的严格定义

Power = 能让公司**持续**赚取**经济利润**（高于资本成本）的**结构性**条件。

3 个关键词：

1. **持续** —— 不是一次性，是 5 年以上
2. **经济利润** —— 不是会计利润，是高于 WACC 的回报
3. **结构性** —— 来自市场结构，不是 CEO 个人能力

**这个定义把 90% 的"护城河"过滤掉了**。比如"管理层优秀"——不算结构性；"产品好"——不算持久。

## 7 种 Power（按强弱排序，我的判断）

### 1. Network Economies 网络经济（最强）

**定义**：每多一个用户/参与者，对所有现有用户的价值提升。

**经典样本**：
- Visa / Mastercard（双边网络）
- Facebook / WhatsApp（直接网络）
- Bloomberg Terminal（90 年代的网络锁定）

**[[network-effects]] 已经深入展开**。Helmer 把网络效应作为最强 power 的核心地位**得到广泛认可**。

### 2. Counter-Positioning 反定位（最微妙）

**定义**：新进入者用一种**结构上让在位者无法模仿**的商业模式攻击。

**经典样本**：
- **Vanguard vs 主动管理**：Bogle 1976 推出指数基金 0.2% 费率。主动基金不能跟，因为：跟了 → 现金流崩溃 → 高薪员工流失 → 公司死亡。**结构性不可模仿**。
- **Netflix vs Blockbuster**：邮寄 DVD 让 Blockbuster 无法跟——他们的 60% 收入来自滞纳金。**取消滞纳金 = 自杀**。
- **Tesla vs 传统车**：电动车架构让传统车厂转型困难——他们的 50% 利润来自内燃机相关零件 + 经销商网络。

**为什么是"反定位"**：不是更好的产品，是**更好的商业模式 + 在位者不能跟随**。

**Counter-positioning 是 Helmer 的最大原创贡献**——其他 power 框架没有这个概念。

### 3. Cornered Resource 独占资源

**定义**：独占某个稀缺、关键、不可复制的资源。

**经典样本**：
- **Pixar 1996-2006**：John Lasseter + 9 个核心动画师（"Brain Trust"）的人才组合
- **Apple 2000-2020**：Steve Jobs + Tim Cook 供应链 + Jony Ive 设计——一旦失去就难复制
- **PD-1 / PD-L1 抗肿瘤药**：早期专利持有者（默沙东 Keytruda）拿到独占
- **稀土永磁**：中国控制 ~80% 全球稀土加工

**独占资源最难维持**——一旦关键人才离开 / 专利到期 / 法规变化，power 立刻消失。

### 4. Scale Economies 规模经济

**定义**：单位成本随规模下降。

**经典样本**：
- Walmart（采购规模 → 单位成本低 → 价格优势 → 销量大 → 采购规模更大）
- Costco（同样机制）
- AWS（数据中心摊薄成本）
- 半导体（TSMC 7nm 投入 $200 亿，只有最大规模才能摊薄）

**规模经济的强弱看"规模相对优势"**：
- Walmart vs Target = 1.5 倍销售 → 弱
- AWS vs 第三大云 = 3 倍以上 → 中等
- TSMC vs 第二大代工 = 5 倍以上 → 强

### 5. Switching Costs 转换成本

**定义**：用户从 A 切换到 B 的成本高（金钱 + 时间 + 风险）。

**经典样本**：
- 企业软件（SAP、Oracle、Salesforce）：换系统 = 半年项目 + 千万投入 + 业务风险
- 银行账户：换银行 = 重新设置 50 个自动转账
- 编程语言生态（一旦学了 Python 不愿换 R）

**Switching costs 最适合 B2B 软件投资**——这是为什么企业 SaaS 公司估值倍数高。

### 6. Branding 品牌

**定义**：消费者愿意为同样产品付溢价的认知优势。

**经典样本**：
- Coca-Cola（每年涨价 5% 而需求不变，[[1988-coca-cola-buffett]]）
- Hermès（爱马仕铂金包等候名单 = 极端品牌力）
- Tiffany、LV、Rolex

**品牌的局限**：
- 大多数品牌没那么强（Buffett 反例 Anheuser-Busch 啤酒品牌弱化）
- 年轻一代品牌忠诚度低
- 一次大丑闻可以毁掉几十年品牌（参考几个汽车召回案）

### 7. Process Power 流程优势

**定义**：内部流程 + 组织能力的复合结果，**多年累积，难以模仿**。

**经典样本**：
- **Toyota Production System** 1950s 起累积，全球丰田大学但其他车厂复制不出
- **Apple 供应链管理**（Tim Cook 1998 加入后建立）
- **Costco 选品 + 会员制**（30 年迭代的产品组合）
- **Berkshire 自己的"资本配置"流程**（Buffett 60 年的 capital allocation 系统）

**Process Power 是最被低估的 power 类型**——因为它**看不见、摸不着**，只能通过结果（持续高 ROIC）反推。

## Helmer 的 3 个时间阶段

Power 不是静态属性，是**动态过程**：

| 阶段 | 特征 | 投资人怎么看 |
| --- | --- | --- |
| **Invention（发明）** | 公司发现一个新的 power 来源 | 高风险高回报；早期 VC 阶段 |
| **Takeoff（起飞）** | Power 开始累积，竞争对手反应不及 | 最佳投资时机；公开市场早期 |
| **Stability（稳定）** | Power 已建立，公司主导市场 | 稳定但回报有限；Buffett-Munger 阶段 |

**Buffett 90% 投资在 Stability 阶段**。Vitalik 创立以太坊在 Invention 阶段。a16z 投早期项目主要在 Takeoff 阶段。

**这个时间维度让 7 Powers 比 Porter 更有用**——它告诉你**什么时候买 + 期望什么回报**。

## 一个具体例子：分析 NVIDIA 2024

用 7 Powers 框架分析 NVDA：

| Power | 强度 | 证据 |
| --- | --- | --- |
| Network Economies | **强** | CUDA 生态、开发者数百万 |
| Counter-Positioning | 弱 | AMD 可以做同样的事，没有结构阻挡 |
| Cornered Resource | **强** | 黄仁勋 + 工程师团队 + TSMC 7nm/5nm 独家 |
| Scale Economies | 中 | 量大但 AMD/Intel 也能达到规模 |
| Switching Costs | **强** | 客户用 CUDA 写的代码切换成本极高 |
| Branding | 弱 | B2B 市场品牌力有限 |
| Process Power | 中 | 设计能力强但可被复制 |

**结论**：NVDA 有 3 个强 power（Network + Cornered + Switching），这是它高估值的基础。

**风险**：Cornered Resource（CUDA 生态）可能被开放标准颠覆（PyTorch、JAX 等抽象层）。

**这种分析是 7 Powers 框架的实操价值**——比"NVDA 牛"具体 100 倍。

## 反面 / 边界

### 1. 7 Powers 也会被颠覆

参考 [[christensen-innovators-dilemma]]——任何 power 在足够大的技术 / 商业模式变革下都可能消失。

例：Bloomberg Terminal 的网络经济在 AI 时代可能减弱（LLM 直接处理金融数据）。

### 2. 多个 power 叠加 = Lollapalooza

最强公司有多个 power 同时存在：
- Apple = Brand + Switching Costs + Network (App Store) + Scale + Process
- 这就是 [[lollapalooza]] 在竞争分析里的应用

### 3. Helmer 框架不擅长预测时点

它告诉你**哪些公司有 power**，不告诉你**什么时候买卖**。

### 4. Counter-Positioning 容易事后归因

很多人事后说"X 是反定位"——但事前看可能只是普通新进入者。**真正的反定位需要结构性证据**。

### 5. Process Power 难以量化

往往只能事后看 ROIC 反推。**这意味着 Process Power 标的是"高 ROIC 持续公司"的代名词**。

## Naval / Patrick Collison / Helmer 自己的投资

Helmer 创立 Strategy Capital，1999-2024 跑赢 S&P 500 显著。**专投有 Power 的公司**。

Naval 在多次播客推荐 7 Powers 是"投资人必读"。

Patrick Collison (Stripe 创始人) 把 7 Powers 框架公开做内部战略讨论的工具。

**这个框架的"市场检验"已经完成**——它是 2016 后投资分析的事实标准。

## 给投资者的启示（5 条）

1. **每个 watchlist 标的写下 7 Powers 评分**——这一招让你立刻看清 power 类型 + 强度。

2. **优先 Counter-Positioning + Network + Cornered Resource**——这 3 个最强 power 类型。

3. **拒绝"品牌 + 规模"作为唯一 power**——很多公司只有这两个，但都是相对弱的 power。

4. **看时间阶段**：Stability 阶段公司 = 稳定回报；Takeoff 阶段 = 高潜回报。**不要在 Invention 阶段重仓**。

5. **Power 不是永久的**——结合 [[christensen-innovators-dilemma]] 视角持续检查 power 是否在被侵蚀。
