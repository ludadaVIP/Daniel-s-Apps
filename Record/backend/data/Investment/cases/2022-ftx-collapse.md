## 一句话核心

FTX 2019 年由 Sam Bankman-Fried（SBF）创立，2 年内成全球第二大加密交易所、2022 年估值 **$320 亿**、SBF 个人净资产 $260 亿（30 岁全球最年轻自做亿万富翁之一）；但 2022-11-08 一周内**崩盘** —— SBF 把 FTX 客户存款**秘密借给** Alameda Research（他控制的对冲基金）用于自营 + 个人花费；**100 万用户损失 $80 亿**，SBF 2023 年判 25 年监禁；这是 [[2022-luna-terra-collapse]] 之后第二只**直接摧毁整个加密寒冬**的关键事件。

## 背景：SBF 的传奇起点

### 早期背景

Sam Bankman-Fried（1992 年生）：
- MIT 物理 + 数学双学位
- Jane Street 量化交易员（2014-2017）
- 接受 effective altruism (EA) 哲学（理性 + 慈善）

### EA 哲学影响

SBF 公开宣称投身金融是为了"earn to give"（赚钱来捐）：
- 计划赚 $1000 亿
- 全部捐给"高影响力"慈善
- 这给了他**巨大道德光环**（[[halo-effect]]）

**实际**：他用慈善叙事掩盖野心 + 后来用客户钱"慈善捐款"（也是欺诈）。

### Alameda Research（2017）

SBF 2017 离开 Jane Street，创立 Alameda Research（加密对冲基金）：
- 利用加密市场不同交易所价差套利
- 早期获巨大利润（"日本溢价"等）
- 1 年内净值从 $0 到 $1 亿

### FTX（2019）

2019 年 5 月 SBF 创立 FTX—— 一个加密衍生品交易所：
- 创新产品（杠杆代币、永续合约）
- 用户体验好
- 营销激进

**爆发性增长**：
- 2019 末：$50 亿日交易量
- 2021 末：$200 亿日交易量
- 2022 中：**全球第二大加密交易所**（仅次于 Binance）

## SBF 的"圣徒"包装

SBF 故意打造**反硅谷富豪**的形象：
- 穿 T 恤 + 短裤（不穿西装）
- 蓬乱头发
- 睡在 bean bag（豆袋椅）
- 玩 League of Legends（电竞游戏）
- 公开承诺捐光所有钱

**结合 EA 哲学** → 媒体把他包装成 "**Robin Hood of Crypto**"：
- 不是为自己赚钱
- 是为了拯救世界

**与 Bezos / Musk 的"霸道"形象形成对比** —— SBF 显得"接地气 + 利他"。

## 政治影响 + 监管"光环"

SBF 用部分财富做**大规模政治捐款**：
- 2022 选举周期捐 ~$4000 万（民主党 + 少量共和党）
- 第二大民主党金主（仅次于 George Soros）
- 与多名国会议员、SEC 主席 Gary Gensler 建立关系
- 与监管者**频繁会面** —— 包装成"合规先锋"

**这给 FTX 巨大"监管光环"**——很多人觉得"SEC 都和他熟，肯定合规"。

但**实际**：这是 SBF 的**烟幕弹** —— 用政治影响阻止针对 FTX 的调查。

## 欺诈机制（3 大支柱）

### 支柱 1：FTX 客户存款 → Alameda 借款

FTX 是**交易所**——理论上**用户钱归用户**，FTX 只保管。

**SBF 的安排**：
- FTX 用户存款（$80 亿+）
- **秘密转给** Alameda Research（SBF 控制的对冲基金）
- Alameda **用客户钱做激进自营**
- 包括加杠杆 + 投资 LUNA / Aave / 失败创业

**这是直接挪用** —— 类似银行用储户钱炒股票（非法）。

### 支柱 2：FTT 代币的循环套利

FTX 发行自己的代币 FTT：
- FTT 持有人获 FTX 平台手续费回扣
- SBF 鼓励 Alameda **大量持有 FTT**
- FTT 作为 Alameda 资产 → 推高 Alameda 估值
- Alameda 用 FTT 作抵押借更多钱

**问题**：FTT 流动性极差 → **市场价不能真实变现**。
**如果有人抛售 FTT** → 价格暴跌 → Alameda 抵押品归零 → 全盘崩溃。

### 支柱 3：复杂关联交易掩盖

SBF + Caroline Ellison（Alameda CEO + 前女友）+ Nishad Singh（FTX 工程师）等少数人**共同设计**：
- 隐藏的 Alameda 账户在 FTX 有"无限信用线"
- 监控系统**忽略** Alameda 的负余额
- 财务报表**永远不和审计**对账

**没有外部审计师** —— FTX 一直拒绝大型四大会计师审计。

## 5 个真正的因果链

### 因果 1：Sequoia 等顶级 VC 失败

Sequoia 投 $210M 进 FTX——基于：
- SBF 的"天才"叙事
- EA 哲学的"道德"光环
- 增长数字
- 但**未做真正尽调**

Sequoia 著名的"SBF 写一本书 while pitching us" 故事 —— 他们**忽视**这种异常行为（创始人在融资演讲时分心打游戏）—— **反而把它解读为"天才"**。

[[halo-effect]] 极端版。

### 因果 2：政治影响阻止调查

SBF 的政治捐款 + 监管者关系让 FTX 多年**免于真正调查**。
**SEC 主席 Gary Gensler 多次见 SBF** —— 但没启动针对 FTX 的执法。

[[incentives]] 视角：监管者的激励是"避免错抓创新者"，不是"主动调查热门公司"。

### 因果 3：LUNA 崩塌的连锁

[[2022-luna-terra-collapse]] 让 Alameda 亏 $30+ 亿：
- 之前 Alameda 持有 LUNA / UST 大量头寸
- LUNA 归零 → Alameda 资产负债表洞
- SBF 决定用**FTX 客户钱填洞**
- **这是欺诈的真正触发点**

### 因果 4：FTT 抵押的反身性

Alameda 用 FTT 作抵押 → FTT 必须涨：
- SBF 控制 FTT 流通量
- 大部分锁定在 Alameda + FTX 内部
- 公开市场 FTT 仅占总量小部分

**一旦有人公开卖 FTT**（CZ 11-06）→ 价格暴跌 → Alameda 抵押品归零 → 系统性挤兑 → 死亡螺旋。

[[reflexivity]] 加密版。

### 因果 5：内部 close-knit 团队的共谋

SBF + Caroline Ellison + Gary Wang + Nishad Singh：
- 都是 SBF 高中 / Jane Street 同事
- 都住在 Bahamas 同一公寓
- "Polycule"（多元恋爱关系）
- 决策**完全在小圈子**，无外部检查

**这是 [[chestertons-fence]] 反例** —— 没有正常公司治理结构（独立董事 / 审计 / 风控）—— 但**没人质疑**。

## 这件事最深刻的 5 个教训

### 教训 1：EA / 慈善光环是**最强**的伪装

SBF 用"earn to give"的道德叙事让人**完全不质疑他的动机**：
- "他不是为自己赚钱"
- "他在拯救世界"
- "怎么可能是骗子"

**实际**：他用"慈善"获取**最高道德权威** + 然后用客户钱"慈善"（也是欺诈）。

**Lesson**：任何过度的道德 / 慈善包装 = **反向警觉**。

### 教训 2：拒绝审计 = 直接退出

FTX 拒绝大型四大审计——**任何加密交易所拒绝审计应该立刻退出**。

[[2022-luna-terra-collapse]] 的 Anchor 也是同样问题。

### 教训 3：交易所 + 自营基金的利益冲突 = 致命

任何同时运营 **交易所 + 对冲基金** 的实体都有**结构性利益冲突**：
- 交易所看到客户头寸 → 自营可以抢跑
- 客户存款 → 可被挪用

**银行业 1933 Glass-Steagall 法案就是为分离这两者** —— 但加密世界没有这种监管。

### 教训 4：媒体光环 = 反向信号

任何创始人**被 Time / Forbes / Fortune 多次封面** —— 反向警觉。

Holmes（Theranos）、SBF、Neumann（WeWork）—— 都被反复封面 → 然后崩塌。

**真正持续优秀的人**（Buffett / Munger / Bezos）**不会反复登封面**。

### 教训 5：内部 close-knit 团队 = 治理风险

任何公司核心团队全是**多年朋友 + 共同生活** —— 治理风险极高。
- 没有外部声音
- 没有 Devil's Advocate
- 集体合理化

**对策**：评估公司 board / executive 多样性。

## 反面 / 边界

### 1. 不是所有加密都是骗局

BTC（[[bitcoin-as-hard-money]]）/ ETH 仍是真创新。
**对策**：区分 "infrastructure"（BTC / ETH）vs "ecosystem 内的应用"（FTX / LUNA / 大多数 DeFi）。

### 2. 加密交易所有合规的

Coinbase（美国合规交易所）虽然股价波动大但**没欺诈**。
**对策**：评估**监管 + 审计 + 资本充足**——这些是真信号。

### 3. EA / 慈善哲学本身不坏

EA 圈子里**很多真诚的人**。SBF 是 EA 的**叛徒**而非代表。
**对策**：哲学本身中性 —— 看个人执行。

### 4. 顶级 VC 也会错

Sequoia 是顶级 VC—— 但 FTX 投资归零。
**没有"权威 VC = 必胜"的护身符**。

[[authority-bias]] + 不做尽调 = 灾难。

### 5. SBF 之后加密监管加强

2023-2024 SEC / CFTC 大幅加强加密执法。
**短期对加密负面**，**长期对真创新有利**（淘汰骗子）。

## 给投资者的启示（5 条）

1. **过度道德 / 慈善包装 = 反向警觉** —— SBF 是 EA 的极端样本。

2. **拒绝审计 = 直接退出** —— 任何金融机构都适用。

3. **交易所 + 自营基金 = 致命利益冲突** —— 法律分离有原因。

4. **媒体反复封面 = 反向信号** —— Theranos / WeWork / FTX 共同模式。

5. **内部 close-knit 团队 + 无外部声音 = 治理灾难** —— 评估 board 多样性。
