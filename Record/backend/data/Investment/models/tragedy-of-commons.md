## 一句话

Garrett Hardin 1968 年发表的经典论文：当资源是**共有的**（没有明确产权），每个个体**理性最大化**自己使用 → 集体上**毁灭资源**——这就是公地悲剧；它解释了**气候变化、过度捕捞、地下水枯竭、AI 算力 race、监管套利、为什么 ESG 投资难成功**——**它是市场失灵最重要的一种**。

## Hardin 的原始故事（1968）

Garrett Hardin（生态学家）1968 年在 Science 杂志发表的经典文章："**The Tragedy of the Commons**"。

**经典寓言**（基于 William Lloyd 1833 原型）：

一个村庄共用一片牧地。每户人家有牛。

**每个牧民的"理性"决策**：
- 多放一头牛 → 我多得 100% 收益
- 草地被啃多一点点 → 成本由所有牧民分摊 → 我只承担小部分

**结果**：每个理性牧民都多放牛 → 草地被过度啃食 → **最终所有牛都饿死，整个村庄毁灭**。

> "**Each man is locked into a system that compels him to increase his herd without limit—in a world that is limited.**"

—— 每个人都被锁在一个系统里，必须**无限增加自己的牛**——在一个**有限的世界**里。

**这就是公地悲剧**：**个体理性的累积 = 集体非理性**。

## 数学定义

公地悲剧的精确条件：

| 维度 | 公地悲剧 | 私有产权 |
| --- | --- | --- |
| 资源所有权 | 共有 / 无人 | 私有 |
| 使用收益 | **完全归个体**（100%） | 完全归个体 |
| 使用成本 | **分摊给所有**（1/N） | 完全归个体 |
| 个体理性 | 多用 | 平衡使用 |
| 集体结果 | **过度使用 → 毁灭** | 可持续 |

**关键不对称**：收益归个体（私有化），成本归集体（社会化） → 必然过度使用。

## 真实世界的 5 大公地悲剧

### 1. 大气层（气候变化）

- 排放温室气体的**收益**归个体 / 公司 / 国家
- **成本**归全球
- 每个国家理性少减排
- **结果**：气候变化加剧

**这是人类历史上最大的公地悲剧**。

### 2. 海洋鱼类（过度捕捞）

- 每艘渔船多捞鱼 → 我多赚
- 鱼群减少 → 所有人分摊
- **结果**：全球大型鱼类种群下降 90%（1950-2024）

### 3. 地下水

- 每个农民多抽水 → 自己多产
- 地下水位下降 → 大家共同承担
- **结果**：北中国 / 印度 / 加州地下水严重枯竭

### 4. 抗生素抗药性

- 每个人 / 农场过度使用抗生素 → 短期收益
- 细菌产生抗药性 → 全球医疗危机
- **结果**：WHO 警告"后抗生素时代"

### 5. 互联网注意力

- 每个 App 多推送、多通知 → 自己用户停留
- 用户注意力被分散 → 大家共担
- **结果**：用户疲劳，整体满意度下降

## 投资里 5 种 specific 公地悲剧

### 1. AI 算力 / 能源 race

[[2026-W22 周报]] 提到的 AI capex 主题。

每个 hyperscaler 理性扩建：
- MSFT、GOOGL、AMZN、META 都想拿到 AI 领先
- 都签 20 年 PPA 锁定电力
- **集体上**：算力过剩 + 电力短缺 + ROI 不确定

**5 年后可能爆发"算力 race 公地悲剧"**——所有人都过度投资，集体回报低于个体期望。

### 2. 监管套利

每家公司理性"找漏洞优化税务" → 政府收入减少 → 监管收紧 → 所有公司面临更严监管 → 包括优秀公司。

**反例**：FTX 利用监管套利 → 引发全行业监管打击 → 即使合规交易所也受影响。

### 3. 高频交易 / 流动性提供

每家 HFT 公司理性优化速度 + 算法：
- 早期：少数 HFT 提供流动性 → 收益高
- 中期：更多 HFT 涌入 → 利润摊薄
- 末期：流动性"提供"变成"抢跑" → 普通投资者吃亏 → 监管打击

**这是 HFT 行业 2010-2020 的轨迹**。

### 4. ICO / 加密代币发行

[[2014-ethereum-ico-mania]]：

每个创业者 / 项目方理性 ICO 募资：
- 项目方：免费的钱
- 投资人：FOMO
- 监管：滞后

**集体结果**：90% 项目归零 → 整个加密行业信誉受损 → 包括好项目（[[bitcoin-as-hard-money]]）受拖累。

### 5. 互联网媒体的"标题党"军备竞赛

每个媒体理性优化点击率 → 标题越来越极端 → 内容质量整体下降 → 用户对所有媒体的信任度下降。

**结果**：媒体行业整体收入下降，公信力丧失。

## Elinor Ostrom 的反驳（1990 诺奖）

Elinor Ostrom（2009 年诺贝尔经济学奖，第一位女性获奖者）—— 实地研究全球数百个共有资源案例。

**她的发现挑战 Hardin**：**很多社区成功管理公地，没有悲剧**。

她列出**8 个原则**让公地可持续：

1. **明确边界**（谁能用，谁不能）
2. **规则适应当地**（不是顶层强制）
3. **集体决策机制**
4. **有效监控**
5. **分级惩罚**
6. **冲突解决机制**
7. **被外部承认的自治权**
8. **嵌套治理结构**

**例**：瑞士阿尔卑斯山的共有牧场——500 多年可持续放牧。日本传统林业。土耳其的渔民社区。

**Ostrom 的洞察**：**Hardin 的悲剧不是必然——是没有治理结构时的结果**。

**关键**：私有产权 + 政府监管 + 社区自治 + 文化规范 = 4 种"避免公地悲剧"的工具。

## 投资里的避免方法

### 1. ESG 投资

理论上 ESG 投资让公司**内化外部性** → 减少公地悲剧。

**实际**：
- 部分有效（清洁能源投资增加）
- 部分无效（greenwashing 普遍）
- 长期效果未明

### 2. 碳定价

让排放有价格 → 内化大气公地。

- EU 碳交易体系
- 加拿大碳税
- **关键**：定价水平要够高

### 3. 监管 / 政府介入

- 渔业配额（个体可转让配额 ITQ）
- 大气污染限制
- 反垄断

### 4. 私有化

把公地变成私有 → 个体承担全部成本。

例：肯尼亚把野生动物保护权给当地社区 → 反偷猎效果改善。

### 5. 社区自治

Ostrom 模式：让使用者自己制定规则。

**最难但最持久**。

## 给投资者的具体启示（5 条）

### 1. 警惕"全行业过度投资"

如果一个行业**所有玩家**都在大规模投资：
- AI 算力（2023-2024）
- 共享单车（2017-2018 中国）
- WeWork-like 联合办公（2018-2019）
- SPAC（2020-2021）

→ **公地悲剧风险高**——集体过度投资 → 整体回报下降。

### 2. 押注"避免悲剧"的工具

- 投碳交易公司
- 投监管科技（RegTech）
- 投资源管理技术（淡水、稀土、电池回收）

**这些是"反公地悲剧"产业**——长期需求强劲。

### 3. 拒绝"集体疯狂"的资产

任何"所有人都在买" + "明显非可持续" = 公地悲剧前夜。

[[2022-luna-terra-collapse]]、[[2014-ethereum-ico-mania]] 都是公地悲剧的变体。

### 4. 长期投资 + Ostrom 思维

投资有**长期治理结构**的公司 / 系统：
- 强公司治理
- 长期股东文化
- 系统化资本配置

这些是 Ostrom 原则在公司层面的应用。

### 5. 个人组合也有公地悲剧

如果你的不同标的都暴露在**同一系统性风险**（如全部美国科技股）：
- 个体上每个都"理性"
- 集体上你的组合在那个系统崩溃时全损

**对策**：真分散 + Ostrom 风格"边界设计"（最大单一暴露 X%）。

## 反面 / 边界（5 个陷阱）

### 1. 不是所有共有资源都悲剧

Ostrom 证明：**有治理就可持续**。

不要用"公地悲剧"作为反对一切公共资源的借口。

### 2. 私有化不是万能

私有化某些资源（基础医疗、教育）反而创造新问题——市场失灵的另一种。

**对策**：根据资源性质选择治理方式。

### 3. 监管也有 [[incentives]] 问题

监管者本身可能被俘获 → 监管失败。
**不是"有监管 = 解决了悲剧"**——需要好监管。

### 4. 公地悲剧的时间维度

短期看似"理性使用" → 长期才暴露。
很多公司 / 投资者**短视** → 即使知道公地悲剧也参与。

[[time-arbitrage]] 提过——时间视角不匹配是核心问题。

### 5. 全球化让公地扩大

气候、海洋这些**全球公地**比传统公地更难治理——**没有全球政府**。

这是为什么气候问题这么难解决。

## 给投资者的启示（5 条）

1. **任何"全行业军备竞赛"小心公地悲剧** —— AI capex、共享经济、SPAC 都是样本。

2. **Ostrom 8 原则可以应用到公司分析** —— 边界清晰 + 治理结构好的公司长期生存更好。

3. **投资"避免悲剧"工具** —— 碳定价、RegTech、资源管理。

4. **拒绝集体疯狂资产** —— 公地悲剧前夜的特征：所有人都在做 + 明显不可持续。

5. **个人组合也要避免公地悲剧** —— 真分散 + 边界设计。
