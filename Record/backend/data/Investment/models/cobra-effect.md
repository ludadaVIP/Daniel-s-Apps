## 一句话

德国经济学家 Horst Siebert 2001 年命名的"眼镜蛇效应"：**为了减少眼镜蛇，英国殖民政府悬赏每条死蛇——结果印度人养蛇杀掉换赏金**；这就是**良意干预的反向后果**——任何"激励某行为"的政策都可能制造**意想不到的恶劣后果**；它是 [[incentives]] 在政策层面的最危险展示，**也是为什么 Munger 反复说"don't optimize for what you can measure"**。

## 原始故事：印度的眼镜蛇

19 世纪英国殖民印度时期，**德里被毒蛇眼镜蛇困扰**。

**英国总督的"理性"解决方案**：
- 每条死眼镜蛇悬赏
- 印度人为了赏金会去杀蛇
- 蛇会减少

**实际发生**：
1. 起初有效——很多野生蛇被杀
2. 不久印度人开始**养殖**眼镜蛇——杀掉换赏金（成本 < 赏金）
3. 政府发现 → **取消赏金**
4. 养蛇人**释放**所有蛇（没价值了）
5. **眼镜蛇数量比政策前更多**

**最终结果**：政策的**完美反向**效果。

**Siebert 2001 年用这个故事命名 "Cobra Effect"** —— 良意但反向的政策后果。

## Soviet 的"钉子工厂"

类似的经典故事（苏联 1930s-1950s）：

**问题**：钉子工厂效率如何衡量？

**版本 1**：按**数量**计 → 工厂生产**大量小钉子**（计件方便，没人要）
**版本 2**：按**重量**计 → 工厂生产**少量超大钉子**（轻松达标，没人要）

**结果**：苏联钉子普遍**不符合实际需求**——因为衡量指标和真实价值脱节。

**这是 [[goodharts-law]] 的孪生**——当指标变成目标，行为就被扭曲。

## Cobra Effect 的本质机制

任何"好意激励"都可能产生反向后果，**原因 3 个**：

### 1. 人会优化"被衡量的"，不是"被期望的"

- 政府想：减少蛇
- 政府衡量：死蛇数量
- 人优化：死蛇数量（不管是野生还是养殖）

**永远有"作弊路径"** —— 而人会找到它。

### 2. 短期成功掩盖长期失败

很多 cobra effect 案例**短期看有效**：
- 死蛇数量增加（前 1 年）
- 钉子产量达标
- KPI 实现

**长期才暴露**：
- 总蛇数增加
- 产品无用
- 系统崩溃

### 3. 第二层 + 第三层效应

[[second-order-thinking]] 的反例：
- 第一阶：减少蛇
- 第二阶：人为养蛇套利
- 第三阶：政策取消 → 蛇被释放
- 第四阶：野生蛇增加

**多数政策制定者只想到第一阶**。

## 现代 Cobra Effect 案例（5 个）

### 1. 美国 Cash for Clunkers（2009）

奥巴马政府 2009 年推出 "Cash for Clunkers"：
- 给折旧老车补贴让车主换新车
- **目的**：环保 + 刺激汽车业

**反向后果**：
1. 大量功能正常的老车被强制销毁
2. **二手车市场供给暴跌**
3. 二手车价格**暴涨 50%+**
4. 低收入家庭**买不起二手车**
5. 总环境收益微不足道（生产新车的碳排放）

**短期"成功"——长期反向**。

### 2. 中国房地产限购（2010s）

很多城市为了"抑制投机"实施限购：
- 单人限购 1 套
- 第 2 套首付高 / 利率高

**反向后果**：
- 大量"假离婚"为了买第二套
- 推高真实需求（恐慌购买）
- 限购解除后房价更快上涨

### 3. 公司股票回购（2010s-2020s）

很多公司大规模股票回购**初衷**是：
- 返还现金给股东
- 提高 EPS（每股收益）
- 优化资本结构

**反向后果**：
- CEO 用回购**人为推高股价** → 自己股权变现
- 真实研发投资被忽视（短期 EPS 第一）
- 工资增长停滞（钱都拿去回购了）

**这是为什么参议员 / 监管者越来越质疑回购**——它部分是 cobra effect。

### 4. ESG 投资强制

强制 ESG 评分**初衷**：让公司更关注环境/社会。

**反向后果**：
- 公司**做表面文章**（光彩报告、ESG 评分游戏）
- 真实变革慢
- 资本被错配（高 ESG 评分≠ 真实贡献）
- 部分公司**反向博弈**——"白领"项目代替真减排

**ESG 行业本身的 [[asymmetric-information]] + cobra effect**。

### 5. 学校"无差生"政策

某些教育系统强制"所有学生通过率"高：
- 教师**降低标准**让所有人通过
- 真实学习被忽视
- 学生未来在职场暴露
- 系统失信

## Buffett & Munger 对 cobra effect 的警觉

Munger 在 USC 1994 演讲：

> "**Never, ever, think about something else when you should be thinking about the power of incentives.**"

—— **永远不要在该想激励时想别的事**。

Munger 经常用 cobra effect 警告：
- 任何看似"好意"的政策 → 想 5 步 second-order
- 任何 KPI 系统 → 想员工怎么作弊
- 任何**奖金挂钩单一指标** → 警觉

### Berkshire 的"反 cobra"实践

Berkshire 对子公司 CEO 的薪酬：
- **不挂钩单一 KPI**
- 不挂钩股价
- 综合长期表现 + 个性化 + Buffett 判断
- 大部分**现金**而非股权（避免短期股价操纵）

**结果**：60 年子公司**基本没出大 cobra effect 事件**——CEO 没有强烈作弊激励。

## 投资里的应用（4 种）

### 1. 警惕"目标过于明确"的公司

任何公司 CEO **太关注**某单一指标：
- 季度 EPS 增长 → 可能砍 R&D / 真投资
- 股价短期表现 → 可能回购代替真增长
- 营收增长 → 可能放宽信贷 / 客户质量

**这些都是潜在 cobra effect 标志**。

### 2. 警惕政策驱动行业

任何**政策强力扶持**的行业：
- 太阳能（2010s）→ 中国大规模补贴 → 产能严重过剩 → 行业利润崩溃
- 电动车（2020s）→ 各国补贴 → 同样轨迹
- 半导体（2020s+）→ 各国"自给自足"补贴 → 长期看可能产能过剩

**短期受益，长期 cobra effect 风险**。

### 3. 警惕投资者保护"过度"

监管 "**保护投资者**" 初衷好，但有时 cobra：
- 信息披露要求过严 → 公司选择私募，散户错过早期机会
- 严格 IPO 标准 → 创业公司用 SPAC 套利（[[2019-wework-narrative-collapse]] 期间反例）
- 复杂监管 → 大公司能应付，小公司被挤出 → 行业集中

### 4. 自己投资行为的 cobra

你自己也可能 cobra effect：
- 设定"年回报 20%" 目标 → 追逐高风险标的
- 设定"每月发现 1 个机会" → 强行 over-trade
- 设定"不亏钱" → 永远不投资

**对策**：目标用"过程质量"而非"结果数字"。

## Munger 的反 Cobra 检查清单

任何激励系统 / 政策 / KPI 实施前问 5 个问题：

### 1. 这个激励**激励什么行为**？

不是字面期望——是**实际诱发**的行为。

### 2. 人能"作弊"达成 KPI 而不实现真目标吗？

如果能 → 高概率会作弊。

### 3. 第二、第三阶后果是什么？

不只是直接效果——3 步后会怎样？

### 4. 谁会被边缘化 / 受害？

任何政策都有受害方——他们会反向博弈。

### 5. 怎么衡量"真目标"而不只是"代理指标"？

很难，但是关键。

**这 5 个问题应用到任何重大决策——能避免 80% 的 cobra effect**。

## 反面 / 边界

### 1. 不是所有干预都 cobra

很多政策**真正解决问题**：
- 安全带强制（救人无数）
- 抗生素发明（救人无数）
- 公共卫生（COVID 疫苗 / 早期防疫）

**对策**：cobra effect 是**一种风险**，不是"所有干预都坏"。

### 2. 不干预也有成本

完全不干预 = 让问题继续：
- 不限购房 = 房价泡沫
- 不监管金融 = 2008 危机
- 不监管药品 = 假药泛滥

**对策**：评估"干预的 cobra 风险" vs "不干预的风险"——选**较小恶**。

### 3. 修正复杂

一旦 cobra effect 出现，**修正可能更糟**：
- 取消蛇赏金 → 释放蛇
- 修订法律 → 利益集团抵抗
- 改 KPI → 员工抗议

**对策**：政策设计**前置**反 cobra 设计——比事后修正容易。

### 4. 衡量真目标的成本

真正衡量"减少蛇 vs 净蛇数量"需要**复杂调查**——成本高。

代理指标（死蛇数量）便宜但有 cobra 风险。

**对策**：高 stakes 决策值得花成本衡量真目标。

### 5. 部分 cobra 是"小恶"

某些 cobra effect 后果有限——可接受。

**对策**：评估"cobra effect 大小" vs"干预收益"——选 net positive。

## 给投资者的启示（5 条）

1. **任何"目标过于明确"的公司 → 警觉 cobra effect** —— 季度 EPS / 股价 / 单一 KPI 都可能扭曲行为。

2. **政策强力扶持的行业 → 长期看可能产能过剩** —— 太阳能 / 电动车 / 半导体的轨迹。

3. **Munger 5 问检查激励系统** —— 你的 watchlist 公司适用，自己投资目标也适用。

4. **目标设定用"过程质量"而非"结果数字"** —— 反 cobra 的个人版本。

5. **任何"良意干预"都可能 cobra** —— 这是 [[incentives]] / [[second-order-thinking]] 的终极警告。
