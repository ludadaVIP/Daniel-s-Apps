## 一句话

**P(假设|证据) = P(证据|假设) × P(假设) / P(证据)** —— 这个 1763 年发现的公式告诉你**如何在新证据出现后理性更新你的信念**；它是 Tetlock 超级预测者比 CIA 分析师准 30% 的数学骨架，也是 Munger 经常说"我应该多用贝叶斯"的内疚之源。

## 历史起点：Thomas Bayes 1763

Thomas Bayes（1701-1761）是英国长老会牧师 + 业余数学家。他生前没有发表这个定理。

去世 2 年后（1763），朋友 Richard Price 整理他的遗稿发现一篇手稿——《An Essay towards solving a Problem in the Doctrine of Chances》。这就是贝叶斯定理的诞生。

**讽刺的是**：Bayes 自己**从未意识到这个发现的重要性**。他终其一生是个默默无闻的牧师。300 年后他的名字成为**统计学最重要的分支**——贝叶斯统计学派。

## 数学公式（必须背下来）

```
P(A|B) = [P(B|A) × P(A)] / P(B)
```

翻译：
- **P(A|B)** = 在 B 发生的前提下 A 的概率（后验概率）
- **P(A)** = 先验概率（你之前的判断）
- **P(B|A)** = 在 A 为真的前提下 B 的概率（似然）
- **P(B)** = B 的总体概率（归一化常数）

简化版（更好记）：
```
后验 = (先验 × 证据强度) / 证据总体频率
```

**关键洞察**：你不应该完全相信新证据，也不应该忽视它。**贝叶斯告诉你具体怎么平衡**。

## 经典悖论：医生测试问题

**问题**：
- 某疾病人群发病率 1%
- 测试准确率 99%（真阳性 99%，假阳性 1%）
- 你测试**呈阳性**，请问你真的得这病的概率是多少？

**直觉答案**：99%。

**贝叶斯答案**：**50%**。

**为什么**？算法：

假设 10,000 人测试：
- 100 人真患病 → 99 人测出阳性（真阳性）
- 9,900 人健康 → 99 人测出阳性（假阳性，1% × 9900）
- **总共 198 人测出阳性，其中只有 99 人真的患病**
- **真阳性 / 总阳性 = 99/198 = 50%**

**这个反直觉结果**说明：**当基础概率（base rate, [[base-rates]]）很低时，即使准确度很高的测试也很难确认**。

这是 Kahneman 反复强调的事实——大多数人**完全忽略基础概率**，只看证据强度。**贝叶斯定理强制你考虑两者**。

## Tetlock 的超级预测者研究

Philip Tetlock（宾夕法尼亚大学心理学教授）1984-2003 跟踪 284 个地缘政治专家的预测——**大多数预测不如随机猜测**。

2011-2015 IARPA Good Judgment Project 找出 "**超级预测者**"——前 2% 的预测准确率比 CIA 分析师高 30%。

**核心发现**：超级预测者的共同方法是**Bayesian thinking**：

1. **永远从基础概率开始**（先验）
2. **小幅调整**而非大幅更新（避免过度反应）
3. **承认不确定性**（用概率而非"是/否"）
4. **持续学习并修正**（每个新证据更新一次）

**关键数字**：
- 超级预测者平均**每次更新调整 5-15%**
- 普通人调整 30%+（过度反应）或 0%（忽视证据）
- 小调整 × 多次 = 收敛到真相

## Buffett & Munger 的隐性应用

Buffett 和 Munger 没明说"贝叶斯"，但他们的方法本质相同：

### Munger 1994 USC 演讲

> "**One of the things you have to do all the time is constantly update your beliefs based on new information. Bayesian thinking, while not always explicitly used, is at the core of all good investing.**"

—— Munger 公开说贝叶斯思维是好投资的核心。

### Buffett 的实操样本

[[1988-coca-cola-buffett]]：
- 先验（1985 前）：可口可乐是普通成熟消费品公司
- 新证据：Goizueta 上任 + 剥离非核心 + 大规模回购 + 1985 New Coke 失败反向证明品牌力
- 后验（1988）：可口可乐是"全球流体消费品早期玩家" + 强护城河
- **行动**：26% 仓位重仓

**这就是贝叶斯更新的实战**——他不是 1985 突然觉醒，是**累积观察 3 年后逐步更新**。

## 投资里 4 个具体应用

### 1. 财报季的更新

每个季度财报来 = 新证据。

**错的做法**：
- 一个好财报 → 完全相信公司
- 一个坏财报 → 完全卖出

**贝叶斯做法**：
- 一个好财报 → 微调 +5% confidence
- 一个坏财报 → 微调 -10%（坏消息一般权重更高）
- 累积 8-12 个季度后形成稳定信念

### 2. 新闻报道的更新

每天大量"X 公司 Y 消息"。

**贝叶斯过滤**：
- 这条消息有多大概率出现**如果公司确实如我所想**？（似然 P(B|A)）
- 这条消息有多大概率出现**无论如何**？（总体 P(B)）
- 如果两者接近，则消息**没有信息价值**——大多数新闻属于此类
- 只有**罕见而具体**的消息有真正的更新价值

### 3. 管理层信件的更新

CEO 信里常有"我们正在做 X 战略"。

**贝叶斯解读**：
- 如果公司真在做 X，CEO 会写 → P(B|A) 高
- 如果公司不做 X，CEO 也可能写（"忽悠")→ P(B|¬A) 也高
- **结论**：信件本身信息价值有限
- 真正有价值的是**资本支出 / 收购 / 招聘**——这些是公司**真做**的证据

### 4. 自己判断的更新

每隔 3-6 个月重新评估自己的持仓 thesis。

**问 3 个问题**：
- 我的 thesis 现在的后验概率是多少？
- 比 3 个月前升 / 降了？
- 升 / 降的具体证据是什么？

**这是把"贝叶斯纪律"嵌入决策日志的方法**。

## Munger 的"内疚"

Munger 在多个场合公开说：

> "**I should use Bayes more. I know it works. I don't use it as often as I should.**"

—— **连 Munger 都觉得自己用得不够**。这告诉我们：**贝叶斯是难的**，不是因为公式难（高中数学），而是因为它要求**持续抑制情绪反应 + 系统化思考**。

## Nate Silver 的应用

Nate Silver（FiveThirtyEight 创始人）2008 总统大选预测对 50 个州里的 49 个。2012 预测 50 / 50。

**他的方法纯粹是贝叶斯**：
- 民调结果 = 证据
- 历史选举模式 = 先验
- 每天小幅更新

**对比**：传统政治评论员 2008 预测错很多——因为他们用直觉、新闻周期、情绪。

**Silver 的洞察**："**信号是稀的，噪音是多的。贝叶斯帮你区分两者**。"

## 反面 / 边界（贝叶斯的 5 个陷阱）

### 1. 先验概率必须合理

贝叶斯公式的输出取决于先验。**输入差 → 输出差**。

如果你的先验是"我朋友推荐的股票成功率 50%"（实际可能 10%），你的贝叶斯计算永远是错的。

**对策**：先验必须用 [[base-rates]]，不是直觉。

### 2. 证据强度估计困难

P(B|A) 和 P(B|¬A) 都需要估计。**对大多数情况你不知道**。

**对策**：极端保守。如果不确定，假设证据强度 = 1（即无信息）。

### 3. 公式精确感欺骗人

算出 P(A|B) = 73.5% 给人精确感——但这取决于多个不精确输入。**真实不确定性 50-90%**。

**对策**：贝叶斯告诉你**方向**和**大致区间**，不是精确小数。

### 4. 人类心理抗拒贝叶斯

[[loss-aversion]] + [[confirmation-bias]] 让我们**抗拒更新**。

具体表现：
- 持有亏损股票时，每个负面消息都被忽略
- 持有盈利股票时，每个正面消息都被强化
- 这是反贝叶斯的行为

**对策**：写决策日志强制更新（决策日志的核心价值）。

### 5. 贝叶斯不是水晶球

它降低犯错概率，**不消除**。Tetlock 超级预测者也只有 70-80% 准确率。**预测本质有不确定性**。

## 实操工具：决策日志的贝叶斯版本

每次重大决策时记录：

```
日期：2026-05-30
标的：CEG (Constellation Energy)
先验 thesis: AI capex 需要长期核电 → CEG 长期受益
先验 confidence: 65%
关键证据 1: MSFT 续签 20 年 PPA → +10%
关键证据 2: 美国电力监管放松 → +5%
关键证据 3: 半导体厂电力需求超预期 → +8%
当前后验 confidence: 88%
```

**3 个月后回看**：
- 哪些证据真出现了？
- 哪些没出现？
- 我的更新合理吗？

**这是把贝叶斯嵌入投资纪律的方法**。

## 给投资者的启示（5 条）

1. **永远从 [[base-rates]] 开始**——这是先验概率。

2. **小幅更新 > 大幅更新**——超级预测者的核心方法。每次 5-15%，不是 50%。

3. **新闻 / 财报 / 管理层信件大多没有信息价值**——只有"如果不真，不会出现"的具体证据才值得更新。

4. **决策日志必须包含 confidence 数字 + 更新历史**——这是把贝叶斯落地的工具。

5. **承认贝叶斯是难的——连 Munger 都没完全用好**。**实操中"系统化更新的习惯"比"精确计算"更重要**。
