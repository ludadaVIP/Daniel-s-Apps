## 一句话

Robert Axelrod 1984 年举办的"重复囚徒困境"计算机竞赛中，**14 位博弈论专家提交策略**——**最简单的 Tit-for-Tat（针锋相对）赢了**：第一轮合作，之后**完全模仿对方上一轮**；这一个 4 行代码的策略证明了**合作可以从纯理性自利中涌现**，**也是 Buffett 60 年商业关系的元规则**。

## Axelrod 的竞赛（1980 + 1984）

Robert Axelrod（密歇根大学政治学家）想理解：**理性自利的个体之间，合作如何可能**？

[[game-theory]] 提过囚徒困境——一次性博弈里，**理性 = 背叛**。

**但现实中重复博弈普遍**——你和邻居、同事、商业伙伴反复打交道。

**Axelrod 的实验**：让全球博弈论专家提交"重复囚徒困境"策略，两两对决 200 轮，看谁累积分数最高。

**第一次竞赛（1980）**：14 个策略参赛。

**冠军**：Anatol Rapoport（数学心理学家）提交的 **Tit-for-Tat**——**只有 4 行代码**。

## Tit-for-Tat 的规则

完整规则极其简单：

1. **第一轮：合作**
2. **之后每一轮：完全模仿对方上一轮**
3. 不主动背叛
4. 被背叛时立即报复
5. 对方合作时立即原谅

**就这样。没有其他逻辑**。

## 为什么 Tit-for-Tat 赢

Axelrod 分析冠军后总结**4 个特征**：

### 1. Nice（友善）

**第一轮合作**。这区分于"恶劣"策略（先背叛）。

**实证**：所有"先背叛"策略都输了。

### 2. Retaliating（报复）

**被背叛立即报复**。不报复的"圣母"策略（永远合作）被恶劣策略**完全榨干**。

### 3. Forgiving（宽恕）

**对方合作立即原谅**。不宽恕（一次背叛 = 永远背叛）的策略导致**双输螺旋**。

### 4. Clear（清晰）

**规则简单透明**。对方能**预测**你的行为 → 双方建立可预测的合作模式。

**这 4 个特征是合作进化的元规则**。

## Axelrod 的第二次竞赛（1984）

第一次竞赛后 Axelrod**公开发表结果** + 邀请第二次竞赛——**让所有人都知道 Tit-for-Tat 赢了**。

**第二次竞赛 62 个策略参赛**——大家**都知道 TFT 是冠军**。

**冠军**：仍然是 **Tit-for-Tat**——再次赢。

**这就是策略稳健性的极致** —— 即使所有人都知道你的策略，你仍然赢。

## 经济 / 商业里的 Tit-for-Tat

### 1. 长期合作伙伴关系

Buffett 60 年的商业关系基本是 Tit-for-Tat：
- 第一次：合作（信任、好价格）
- 之后：基于对方行为调整
- 守信对方 → 继续合作
- 失信对方 → 立即停止

**结果**：建立了 Wall Street 上最强的"声誉资本"——大家**主动**找他做生意。

### 2. 国际贸易

WTO 框架本质是 Tit-for-Tat：
- 你不提关税 → 我不提
- 你提关税 → 我对等报复
- 你撤销 → 我撤销

**1930s 没有这种机制时全球贸易战 + 大萧条**。

**2018-2024 美中贸易战**就是 TFT 在国家层面的展示——但**双方都在测试对方边界**。

### 3. 集体行业自律

某些行业（律所、医疗、咨询）有不成文规则：
- 不挖对方核心员工
- 不诋毁对方品牌
- 不打恶性价格战

**TFT 维持**：一旦有人破规 → 立即报复 → 系统恢复合作 or 双输。

### 4. 加密圈的早期合作

[[2010-bitcoin-early-believers]] / [[2014-ethereum-ico-mania]] 早期：
- 核心开发者协作开源
- 互相 review 代码
- 共同推动生态

**TFT 维持**——直到 2017 ICO 时代很多人破规（推垃圾项目）→ 信任崩溃 → 行业声誉受损。

## Robert Triver 的进化生物学

Robert Triver 1971 提出 "**reciprocal altruism**"（互惠利他主义）—— 解释自然界为什么有合作。

**他的核心论点**：

> "**Cooperation evolves when individuals interact repeatedly and remember each other's behavior.**"

—— **个体重复互动 + 记住对方行为 → 合作进化**。

**自然界样本**：
- 吸血蝙蝠互相分享血食物
- 灵长类动物互相梳毛
- 鸟类互相警告捕食者

**这些都符合 TFT 模式**。**合作不需要"道德" —— 只需要重复博弈 + 记忆**。

## TFT 的弱点：错误连锁

TFT 不完美——**它对"噪音"敏感**。

**场景**：
- 你打算合作
- 但发送过程出错 → 对方收到"背叛"信号
- 对方报复 → 你被"背叛"
- 你报复 → 对方再报复
- **双输螺旋**

**真实世界**：很多商业关系崩溃是这种**误解 + TFT 报复 + 升级**的结果。

## TFT 的改良版本

### 1. Tit-for-Two-Tats（针锋相对，但宽容一次）

被背叛**两次**才报复。

**优势**：减少"噪音"导致的错误报复。
**劣势**：被"周期性背叛"的对手榨干。

### 2. Generous Tit-for-Tat

被背叛后**有概率（10%）原谅**——不是 100% 报复。

**优势**：打破恶性循环。
**劣势**：被"恶意试探"的对手利用。

### 3. Win-Stay, Lose-Shift

**胜利时继续当前策略**；**失败时切换**。

**简单但有效**——在某些环境胜过 TFT。

## 投资里的 4 种应用

### 1. 长期商业关系建模

任何**长期重复**商业关系：默认 TFT 心智。

- 首次：合作
- 对方守信：继续合作
- 对方失信：明确反馈 → 调整关系

**关键**：不要"过度宽容"也不要"过度报复"。

### 2. 选择合作伙伴

**首要选择标准**：对方是否**理解并使用 TFT 思维**？

- 对方是 "Always Defect"（永远背叛）？远离
- 对方是 "Always Cooperate"（圣母）？危险（会被恶人榨干 → 间接影响你）
- 对方是 TFT 思维？**这是最好合作伙伴**

### 3. 短期博弈 vs 长期博弈

**警觉 "一次性博弈" 思维者**：
- 旅游区餐厅：你不会再来 → 服务差
- 套现退出的创业者：不再关心声誉 → 可能欺骗
- 离职员工最后一周：不再关心未来 → 可能搞事

**对策**：识别"一次性博弈"对方 → 不信任。

### 4. 自己的声誉资本

按 TFT 原则建立**长期声誉**：
- 守信
- 报复不公（但理性、对等）
- 宽容真诚错误
- 行为可预测

**这是 Buffett 60 年的财富 + 声誉双复利的根源**。

## 反面 / 边界

### 1. TFT 不是普适

某些环境不适合 TFT：
- 一次性博弈
- 对方无记忆
- 信息严重不对称

**对策**：根据情境选择策略。

### 2. 过度报复风险

被背叛立即报复**可能升级冲突**：
- 国际冲突
- 商业敌对
- 个人关系破裂

**对策**：报复要**对等**——不要升级。

### 3. 宽容也有边界

过度宽容 = 被持续背叛者榨干。

**对策**：3 次明确背叛 → 永久断绝（不是 TFT 的本意，但现实需要）。

### 4. 噪音问题严重

[[asymmetric-information]] 让 "背叛 vs 真诚错误" 难以区分。

**对策**：**沟通**——背叛前先问对方真实意图。

### 5. 文化差异

某些文化（如东亚）更强调"面子" → TFT 的报复阶段被视为"不给面子"。

**对策**：考虑文化背景调整 TFT 应用。

## 给投资者的启示（5 条）

1. **长期商业关系按 TFT 心智运行** —— Buffett 60 年的声誉资本来源。

2. **首次默认合作，对方失信明确报复** —— 4 大特征的核心。

3. **警觉"一次性博弈"思维者** —— 不要给他们信任。

4. **建立可预测的声誉** —— Clear 是 TFT 4 大特征之一。

5. **报复要对等，宽容要快** —— 避免双输螺旋。
