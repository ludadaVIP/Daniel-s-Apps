## 一句话

Paul David 1985 年用 QWERTY 键盘的故事提出"路径依赖"：**早期看似偶然的小选择，因为正反馈机制（学习成本、网络效应、规模经济）被锁定**，**即使后来出现更好的选择也无法取代**；这解释了为什么我们今天还在用 110 年前的 QWERTY、为什么 Windows 击败更好的 Mac、**为什么投资里"早期赢家"经常永久赢家**。

## QWERTY 的经典故事（David 1985）

Paul David（斯坦福经济学家）1985 在 American Economic Review 发表"Clio and the Economics of QWERTY"——**路径依赖理论的奠基论文**。

**故事**：

QWERTY 键盘布局 1873 年由 Christopher Sholes 发明。

**关键事实**：这个布局**故意设计成低效**——把常用字母分散，**防止机械打字机按键卡住**。

**1900s 后**机械问题已解决。1936 年 August Dvorak 发明 Dvorak 布局——**实验证明效率高 30-50%**。

**但 QWERTY 仍然主导**——为什么？

**David 的解释（4 个锁定机制）**：

### 1. 训练成本

亿万人已学会 QWERTY。**重学 Dvorak 需要 6-12 个月**。个人 ROI 低。

### 2. 习惯惯性

成年人不愿改变物理习惯。

### 3. 网络效应

学校教 QWERTY → 学生只会 QWERTY → 老师只会 QWERTY → 学校继续教 QWERTY。

### 4. 投资沉没

键盘制造商已投资 QWERTY 工艺；改 Dvorak 需重新投资。

**结果**：**110 年后我们仍然用一个故意低效的布局**——纯粹因为路径依赖。

## Brian Arthur 的扩展（1994）

W. Brian Arthur（圣塔菲研究所经济学家）在《Increasing Returns and Path Dependence》（1994）扩展了 David 的理论。

**Arthur 的核心论点**：

> "**In markets with positive feedback mechanisms, small early events can lock in a particular outcome, even when other outcomes would have been more efficient.**"

—— **在有正反馈机制的市场，小的早期事件能锁定特定结果——即使其他结果更高效**。

**Arthur 列出 4 种正反馈源**：
1. **网络效应** (越多人用越好用，[[network-effects]])
2. **规模经济** (越大越便宜)
3. **学习效应** (用得越熟越擅长)
4. **预期** (人们预期 X 会赢 → 选 X → X 真的赢)

**这 4 个机制让"路径依赖"成为现代经济的常态**。

## 5 个商业路径依赖案例

### 1. Microsoft Windows vs Apple Mac

1980s Mac 在多个维度**优于** Windows：
- 用户界面更好
- 稳定性更好
- 设计更精美

**但 Windows 锁定**：
- IBM PC 早期开放架构 → 大量软件
- Windows 95 决定性胜利
- 企业 IT 标准化
- 学校教 Windows
- 应用程序生态

**结果**：Windows 长期市场份额 90%+。

**今天 Mac 复兴**——但花了 30 年 + Apple 的反向创新（iPod / iPhone / iPad），不是 Mac 本身。

### 2. JavaScript

JavaScript 由 Brendan Eich **10 天内**写成（1995 年）—— 充满设计缺陷。

**但**：
- 浏览器默认支持
- 没有竞争者（Java applet 失败）
- 学习成本累积
- 生态建立

**今天**：JavaScript 是全球最广泛使用的编程语言。即使**所有人**承认它有缺陷——**仍主导**。

**反例**：TypeScript 试图修复 JS 缺陷——必须**编译成 JS** 才能运行。路径依赖太强，无法绕过。

### 3. 美元作为储备货币

二战后布雷顿森林体系把美元定为全球储备货币。

**多次挑战**：
- 1970s 日元
- 1990s 欧元
- 2000s 人民币

**美元仍然主导**——70%+ 全球贸易、储备、外汇交易。

**为什么**：
- 学习成本（重新调整全球金融系统）
- 网络效应（流动性最深）
- 制度信任（美国法律 / 监管）
- 预期（人们继续信任 → 美元继续主导）

**这是 [[bitcoin-as-hard-money]] 挑战的根本难度** —— 美元不只是货币，是整套路径依赖。

### 4. 中国互联网生态

中国早期通过监管创造了"国内互联网"：
- 微信 (vs WhatsApp)
- 阿里 (vs Amazon)
- 百度 (vs Google)
- 抖音 (vs YouTube)

**初期看似"被保护"**——但 10-15 年后形成了**完整的路径依赖**：
- 用户习惯
- 商家生态
- 支付系统
- 物流网络

**今天即使开放**——美国公司也很难进入。**路径依赖锁定了中国本土赢家**。

### 5. 加密赛道的赢家锁定

[[bitcoin-as-hard-money]] / [[2014-ethereum-ico-mania]]：

BTC + ETH 在 2010-2014 早期锁定第一第二。

后续"以太坊杀手"（EOS / Cardano / Solana / Polkadot）大多失败——**不是因为技术差**，而是**路径依赖让 ETH 锁定**：
- 开发者生态
- 工具链
- DeFi 协议
- NFT 标准
- 用户钱包

**Solana 算半个例外**——但即使它在某些维度更快，**整体生态远落后 ETH**。

**路径依赖在加密里证明了 [[network-effects]] 的力量**。

## 投资应用（5 种）

### 1. 偏好"早期赢家"

任何**正反馈机制强**的行业：**早期赢家几乎永久赢**。

- 操作系统：Windows 80s 锁定
- 搜索：Google 2000s 锁定
- 电商：Amazon / Alibaba 锁定
- 社交：Facebook 2004 锁定

**投这些"已锁定"的赢家**——长期回报远超试图押"颠覆者"。

### 2. 警惕"颠覆"叙事

[[christensen-innovators-dilemma]] 的反面：**路径依赖让大多数"颠覆"失败**。

- 大多数"Google killer" 失败
- 大多数"Facebook killer" 失败
- 大多数"Amazon killer" 失败

**只有真正的范式转换**（移动互联网、AI）才能突破路径依赖。**多数"颠覆"是市场幻觉**。

### 3. 利用历史时机

路径依赖意味着**早期时机决定一切**：

- BTC 2010-2013 入场 → 永久红利
- Amazon 1997-2003 入场 → 永久红利
- Tesla 2010-2016 入场 → 永久红利

**入场晚** = 享受不到路径依赖红利。

### 4. 评估护城河时考虑路径依赖

[[helmer-7-powers]] 的扩展：**路径依赖让 power 更持久**。

- 强 power + 强路径依赖 = 几十年护城河（Microsoft / Coca-Cola）
- 强 power + 弱路径依赖 = 几年护城河（很多 SaaS）

### 5. 你自己的路径依赖

**你的投资风格 / 圈子 / 信息源**也有路径依赖：
- 你早期接触哪种投资 → 终身倾向
- 你早期失败的经历 → 终身警惕
- 你早期成功的经历 → 终身偏爱

**警觉**：你的路径依赖可能让你错过结构性变化。

## 反面 / 边界

### 1. Liebowitz & Margolis 的反驳

经济学家 Stan Liebowitz + Stephen Margolis（1990s）质疑 QWERTY 故事——他们认为 Dvorak 优势被夸大。

**他们的论点**：
- 真实效率差异只 5-10%（不是 30%）
- 训练成本超过收益
- 市场实际上**做出了理性选择**

**结论**：**有些"路径依赖"其实是理性结果**——不是真锁定。

**对策**：仔细区分"真锁定"vs"理性选择"。

### 2. 路径依赖不永久

很多看似锁定的事物**最终被颠覆**：
- IBM 大型机 → 被 PC 颠覆
- Kodak 胶片 → 被数码颠覆
- Nokia 手机 → 被 iPhone 颠覆

**路径依赖让颠覆难，不是不可能**。

### 3. 范式转换打破路径依赖

主要范式转换（互联网、移动、AI）能**重置游戏**：
- 桌面互联网赢家不一定是移动赢家
- PC 时代赢家不一定是云时代赢家
- Web2 赢家不一定是 AI 时代赢家

**对策**：识别范式转换 → 押新赢家，而非旧赢家。

### 4. 路径依赖造成低效

不是所有锁定都好。
- QWERTY 让全人类打字效率低
- 公制 vs 英制
- 各种历史遗留制度

**社会效率成本**真实存在——但对投资人**这正是 alpha 来源**（押锁定赢家）。

### 5. 你不能改变路径依赖

试图"重塑行业"基本失败。**Don Quixote 战风车**。

**对策**：理解路径依赖 → 在已锁定的方向**顺势而为**。

## 给投资者的启示（5 条）

1. **强路径依赖行业 = 长期持有的安全 zone** —— Windows / Coca-Cola / Visa / 微信。

2. **大多数"颠覆"失败** —— 路径依赖让在位者赢。

3. **早期时机决定一切** —— BTC 2010 / Amazon 1997 / Tesla 2012 是不可重现的时点。

4. **范式转换是真正的机会** —— PC → 移动 → AI 每次重置游戏。

5. **你自己也有路径依赖** —— 警惕自己的认知锁定。
