## 一句话

博弈论是**多个理性主体决策互相影响**时的数学——Nash 1950 的均衡概念 + Axelrod 1984 的合作进化研究 + Schelling 1960 的"策略"——它解释了**为什么所有人都按规则做事却得到糟糕结果（囚徒困境）**、为什么**冷战没爆发**、**为什么定价战难以避免**、**为什么投资人不该认为市场是"我一个人在玩"**。

## 历史起点：Von Neumann 1944

John von Neumann（20 世纪最伟大数学家之一）+ Oskar Morgenstern 1944 年出版《Theory of Games and Economic Behavior》—— **博弈论作为正式学科的诞生**。

**核心论点**：经济学不是孤立个体最大化效用——是**多人互动**的博弈。

**Von Neumann 的"零和游戏"分析**奠定了基础——但**真实世界大多数博弈不是零和**。这是 Nash 后来扩展的重点。

## John Nash 的革命（1950）

John Nash（《美丽心灵》的主角，21 岁博士论文）扩展了博弈论到**非零和**：

**Nash 均衡** (Nash Equilibrium)：
- 一组策略，使**每个玩家**在**其他人策略不变**的情况下**没有动机改变**
- 不一定是"最好"——只是"稳定"

**关键洞察**：均衡可以是**集体次优**——所有人都"理性"但集体糟糕。

**Nash 1994 年共享诺贝尔经济学奖**（和 John Harsanyi + Reinhard Selten）。

## 囚徒困境（最经典样本）

**Albert Tucker 1950 创造的故事**：

两个罪犯被抓，警察分开审讯：
- **都不招** → 各判 1 年（轻罪）
- **一个招另一个不招** → 招的释放，不招的判 10 年
- **都招** → 各判 5 年

**支付矩阵**：

| | B 不招 | B 招 |
| --- | --- | --- |
| **A 不招** | (-1, -1) | (-10, 0) |
| **A 招** | (0, -10) | (-5, -5) |

**Nash 均衡**：**两人都招**（各 -5 年）。

为什么？因为对每个人：
- 如果对方不招：我招 (0) > 我不招 (-1)
- 如果对方招：我招 (-5) > 我不招 (-10)
- **不管对方做什么，我招都更好**

但 **(招, 招)** 是 (-5, -5)，**而 (不招, 不招)** 是 (-1, -1) 集体更好。

**理性决策导致集体次优**——这是博弈论最深刻的洞察。

## 投资里的囚徒困境

### 1. 卖出泡沫资产

[[2014-ethereum-ico-mania]] 顶部：
- 每个人**都知道**是泡沫
- 但**如果我一个人卖**，泡沫继续 → 我错过最后涨幅
- 如果**大家都卖**，崩盘
- → **大家都不卖**（Nash 均衡）→ **更晚的崩盘**

这就是为什么**叙事谬误**（[[narrative-fallacy]]）能维持那么久——所有人都看清但都不动。

### 2. 价格战

零售业 / 航空业 / 餐饮业：
- 大家"理性"应该不打价格战 → 集体利润高
- 但每个人**理性降价**抢市场份额
- → 价格战 → 集体利润低

**Buffett 多年警告航空业是糟糕生意**——典型囚徒困境。

### 3. R&D 军备竞赛

[[red-queen-effect]] 是博弈论视角的应用：
- 每家公司"理性"加大 R&D
- 集体上 R&D 支出爆炸 → 利润率下降

### 4. 内幕信息使用

- 如果只有 X 用内幕信息 → 他赚翻
- 如果所有人都用 → 监管打击 + 市场失信
- → 法律强制要求"不用"——监管打破囚徒困境

## Axelrod 的合作进化（1984）

Robert Axelrod（密歇根大学政治学家）1984 年出版《**The Evolution of Cooperation**》—— 一本改变多领域的书。

**他的实验**：
- 让 14 位博弈论专家提交"重复囚徒困境"策略
- 各策略两两对决 200 轮
- 谁累积分数最高？

**结果**：最简单的策略 **Tit-for-Tat（针锋相对）** 赢了。

**Tit-for-Tat 规则**：
1. 第一轮合作
2. 之后**模仿对方上一轮**
3. 永远准备宽恕（一旦对方合作）

**为什么 Tit-for-Tat 赢**：
- 不挑衅（开始合作）
- 不被欺负（被背叛立即报复）
- 不记仇（对方合作立即原谅）
- 简单透明（对方知道你的规则）

**这给出了一个深刻洞察**：**在重复博弈里，合作可以从纯理性自利中涌现**。

## 应用到投资和商业

### 1. 长期合作 vs 一次性博弈

**一次性博弈**（不会再见到对方）：**理性是背叛**——这就是为什么旅游区餐厅普遍质量差。

**重复博弈**（长期关系）：**理性是合作**——长期合作伙伴诚信高。

**应用**：
- 短期投资 = 一次性博弈 → 内部人 / 短期玩家可能欺骗
- 长期投资 = 重复博弈 → 大家有动机维持声誉
- → **[[time-arbitrage]] 不仅是时间套利，也是 game theory 优势**

### 2. Buffett 的"声誉资本"

Buffett 60 年建立的声誉：
- 给对手公平价格
- 不耍诡计
- 永远兑现承诺

**博弈论解释**：他在所有交易里玩"长期重复博弈"——一次背叛会摧毁声誉资本。

**结果**：人们**主动**找他做交易，给他**好价格** → 这是 [[opportunity-cost]] 角度的巨大 alpha。

### 3. Schelling 的"焦点"

Thomas Schelling（2005 诺奖）研究"协调博弈"——双方想合作但**不能沟通**时怎么办？

**Schelling 的"焦点"概念**：两个人各自独立猜对方会怎么选 → 会聚焦于**显眼 / 文化默认**的选择。

**例**：让两个陌生人在纽约约见面，不能沟通。多数人选**中央车站正午**——因为这是"焦点"。

**投资应用**：
- 美元为什么是储备货币？焦点
- 比特币为什么是加密 #1？焦点
- 苹果为什么是科技龙头？焦点

**焦点带来强护城河**——很难取代。[[network-effects]] + Schelling focal point = 强结合。

## 反 Nash 均衡：如何打破"集体次优"

如果 Nash 均衡是集体次优，怎么破？

### 1. 沟通 + 承诺

囚徒困境如果两人可以**事前沟通 + 签约** → 可以达成"都不招"。

商业里：行业协会、契约、长期合同。

### 2. 重复博弈

Axelrod 证明：**重复让合作涌现**。所以建立长期关系。

### 3. 改变支付结构

- 加重背叛成本（监管罚款）
- 减轻合作成本（补贴）
- → 改变 Nash 均衡

[[incentives]] 视角：**改变激励就改变均衡**。

### 4. 外部仲裁

引入第三方（监管、法院）打破囚徒困境。

### 5. 信息透明

如果所有人**都看得到**别人是否合作 → 背叛成本提高 → 合作概率提高。

## 投资里 4 种具体应用

### 1. 想象其他玩家的反应

任何投资决策前问：
- **其他人会怎么反应**？
- **如果所有人都做我想做的**，结果如何？

这是从单人决策到博弈思维的升级。

### 2. 偏好"长期重复博弈"标的

- 强声誉公司（Berkshire、Costco）
- 长期客户关系业务（医疗、企业 SaaS）
- 行业领导（Schelling focal point）

**避免**：
- 一次性博弈业务（旅游、短期合同）
- 弱客户黏性

### 3. 看穿"叙事博弈"

很多投资圈"流行"叙事是博弈均衡：
- 所有 KOL 都推 X → Nash 均衡（流量需要追热点）
- 所有分析师都看好 Y → Nash 均衡（不看好会丢工作）
- 所有散户都买 Z → 反身性 + Nash

**看穿这些 = alpha**。

### 4. 用焦点 + 网络效应识别赢家

Schelling 焦点 + 真实网络效应 = 难以颠覆的护城河。

样本：
- [[bitcoin-as-hard-money]] 是加密的焦点
- 微信是中国通讯的焦点
- Visa 是支付的焦点

## 反面 / 边界（5 个陷阱）

### 1. 不是所有人都"理性"

博弈论假设理性 → 现实里很多人**情绪驱动**（[[loss-aversion]] / [[confirmation-bias]]）。

**对策**：**理解理性框架 + 接受非理性偏差** = 真实模型。

### 2. 完美信息假设

很多博弈论模型假设**完全信息**——现实是不完全（[[asymmetric-information]]）。

**对策**：用 Harsanyi 的"不完全信息博弈"扩展（也是诺奖工作）。

### 3. 多重均衡

很多游戏有**多个 Nash 均衡** → 你不知道实际会去哪个。

**对策**：用 Schelling focal point 帮助预测。

### 4. 博弈论分析成本高

每个决策都做严格博弈分析 = 瘫痪。

**对策**：用直觉 + 关键时刻深度分析。

### 5. 博弈论让你过度分析

知道博弈论后**有些人过度推演对方** → 抑制行动。

Nash 自己晚年说过：他的"美丽心灵"病症部分**因为过度博弈论思维**。

**对策**：博弈论是工具，不是生活方式。

## 给投资者的启示（5 条）

1. **任何决策前问"其他人会怎么反应"** —— 单人决策升级为博弈决策。

2. **偏好长期重复博弈业务** —— Axelrod 证明合作在重复博弈中涌现。

3. **看穿 Nash 均衡的"集体次优"** —— 行业囚徒困境是真陷阱。

4. **Schelling 焦点 + 网络效应 = 最强护城河** —— [[bitcoin-as-hard-money]] / Visa / 微信。

5. **建立声誉资本** —— Buffett 模式。长期博弈里声誉是最大 alpha 来源。
