## 一句话

**每多一个用户加入，对所有现有用户的价值就提高**——这创造了"赢家通吃 + 后来者无法追上"的最强护城河；理解它能帮你识别下一个 Facebook，也能避免下一个 Friendster。

## 5 种主要类型

| 类型 | 例子 | 特征 |
| --- | --- | --- |
| **直接网络** | 电话、微信、WhatsApp | 用户越多越好用 |
| **间接网络** | iOS（App + 用户） | 一边多了带动另一边 |
| **数据网络** | Google 搜索、Tesla 自驾 | 用户多 → 数据多 → 产品好 → 用户更多 |
| **本地网络** | Uber / 美团 | 在**特定城市**内有效 |
| **社交网络** | 抖音、小红书 | 用户带朋友进来 |

## Metcalfe 定律

网络价值 ≈ n²（n 是用户数）。1 个电话没用，10 个电话能打 90 个组合，100 个能打 9900 个——**价值不是线性增长**。

## 一个具体例子：Facebook vs Friendster

Friendster 2003 年是第一个出圈的社交网络，5 个月 300 万用户。但服务器跟不上 + 假账号管理失败 → 用户体验崩溃。

Facebook 2004 年才出现，靠**先封闭哈佛 → 再开放校园 → 最后全社会**的策略，避开了同样陷阱。

10 年后 Friendster 关停，Facebook 估值 5000 亿美元。**同样市场，时机差不到 2 年，结局相差万亿美元。这就是网络效应的赢家通吃。**

## 投资里怎么用

1. **找具有真实网络效应的公司**：先问"**如果用户少一半，产品价值是不是少很多？**"
2. **警惕"伪网络效应"**：很多 SaaS 自称网络效应，其实只是"客户多 → 营收高"——这是**规模**，不是**网络**。
3. **早期识别**：网络效应在临界点之前增长慢，之后陡升。看 DAU/MAU 比率、留存率，比看绝对用户数更准。
4. **理解 [[bitcoin-as-hard-money]] 的核心是直接网络效应**——越多人持有越想持有。

## 反面 / 边界

- **网络效应会反向**：用户离开时也是加速的。Snapchat 或 Twitter 这种平台一旦开始衰败，速度比上升时还快。
- **本地网络不抗跨地区**：Uber 全球，但每个城市要重新攻一遍。Facebook 是真正的全球直接网络，更强。
- **政府监管能切断**：中国的 Facebook = 微信，因为防火墙物理切断了网络效应。
- **多归宿（multi-homing）会侵蚀**：用户同时用 Uber + Lyft，网络效应被稀释。

## 一个反例：Clubhouse 2021

2021 年初 Clubhouse 红极一时，DAU 千万级。看起来有强网络效应——结果 1 年内 DAU 跌 90%+。

**原因**：用户的"网络"其实是少数明星和 KOL——他们一走，普通用户就没理由留。**真正的网络效应需要"普通用户对普通用户也有价值"**，而不是依赖少数节点。

## 落地动作

每次评估平台公司，写下：

1. 网络效应是哪一种？（5 类对照）
2. 临界点过了吗？（看 DAU/MAU 留存）
3. 有没有 multi-homing 风险？
4. 监管会不会切断网络？

4 个问题答好之后再考虑估值。
