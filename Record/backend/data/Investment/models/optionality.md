## 一句话

期权性 = **有限下行 + 无限（或显著大于下行的）上行**——这种"不对称回报结构"是 Buffett、Bezos、Taleb 共同的下注哲学；理解它能让你**对赌赢的机会更主动 + 对赌输的风险更敏感**，**几乎是所有伟大投资人和企业家的元方法**。

## 数学定义（来自金融期权）

期权（option）= 一种合约，**给你权利但没有义务**做某事。

**关键不对称**：
- 最大损失 = 你为期权付的钱（premium）
- 最大收益 = **理论上无限**（如果是看涨期权）

```
Payoff = max(0, S - K) - premium      [Call option]
```

其中 S = 标的价格，K = 行权价。

**这种 payoff 曲线是凸的（convex）**——这就是 "convexity"（凸性）概念的来源。

## 凸性 vs 凹性（视觉化）

```
凸 (Convex)：               凹 (Concave)：
    回报                      回报
     |    /                    |  --___
     |   /                     |       \
     |  /                      |        \
     | /                       |         \
     |/_____ 价格              |_________  价格
     0                         0
```

- **凸 (Convex)**：损失有限，收益放大 = 期权多头
- **凹 (Concave)**：收益有限，损失放大 = 期权空头、加杠杆

**Taleb 的核心论点**：投资人应该追求**凸的 payoff**，避免凹的。

## Buffett 的隐性应用

Buffett 不直接说"期权性"，但他实际上一辈子在用。

### 1. Berkshire 浮存 (Float) 的期权性

[[1972-sees-candies]] 之后 Buffett 持续收购保险公司（GEICO、General Re 等）。

**保险浮存 = 客户先付保费，理赔在未来**——意味着保险公司在赔付前可以**自由使用这笔钱投资**。

**期权性特征**：
- 下行：必须赔付（但有再保险对冲）
- 上行：浮存可以拿去投资 → 复利
- Berkshire 当前浮存 ~$1700 亿 → 每年至少 5% 投资收益 = $85 亿"白送"

### 2. 2008 投资 GS / GE / BAC 的期权性

[[2008-lehman-multidimensional]] 期间 Buffett 拿 10% 优先股 + 认股权证（warrants）。

**结构**：
- **优先股**：10% 稳定股息（下行保护）
- **认股权证**：5 年内行权价 $115 的 GS、$22 的 GE
- 下行：拿 10% 利息（已经赚）
- 上行：股价涨破行权价后**所有涨幅都归你**

**实际结果**（2013 年退出）：
- 优先股股息：$15 亿
- 认股权证行权：$25 亿+
- 总投资 $50 亿 → 累计回报 $100 亿+
- **2 倍回报，几乎零下行风险**

**这是教科书级的期权性投资**。

## Bezos 的"多个小赌注"

Bezos 1997 第一封致股东信明确：

> "**We will make bold rather than timid investment decisions where we see sufficient probability of gaining market leadership advantages. Some of these investments will pay off, others will not, and we will have learned another valuable lesson in either case.**"

**翻译**：押多个"赌错也就赔小钱、赌对就改变公司"的小赌注。

具体样本：
- **AWS（2006）**：内部小项目 → 现在 $1T 业务
- **Kindle（2007）**：风险小 → 创造电子书市场
- **Prime（2005）**：争议大但下行小 → 锁定 2 亿会员
- **失败：Fire Phone（2014）**：投入 $1.7 亿 → 关闭 → 学到很多
- **失败：Echo Show 1.0**：销量平平 → 但 Echo 平台依然成功

**Amazon 至今"失败项目"超过 100 个**。**但每个失败下行有限，每个成功改变公司**——这就是组合层面的期权性。

## Taleb 的"远 OTM 期权"策略

Universa Investments 的核心策略：买**远 out-of-the-money** 期权。

**为什么远 OTM 而不是 ATM**：
- ATM（at-the-money）期权贵
- 远 OTM 期权便宜但凸性极高
- 极端事件时回报 100-1000 倍

**实际数字**（2020 COVID 闪崩）：
- Universa 30 天内回报 ~4000%
- 但 2018-2019 大幅亏损（小亏多数时间）
- 长期年化 +10-20%（含所有小亏）

**这就是期权性的极端样本**——99% 时间小亏 + 1% 时间 100x 收获 = 长期 +20%/年。

## 4 种实操期权性

### 1. 真实期权（金融工具）

- 买入看涨 / 看跌期权（成本明确，回报凸）
- LEAPS（长期期权）：1-3 年到期，避免短期波动
- 适合：对市场方向有信心但需要保护

### 2. 隐性期权（业务结构）

- AWS 早期是 Amazon 的"隐性期权"——可能死可能 1T
- 任何"高赔率小投入"业务都有隐性期权
- 适合：在投资分析时识别公司的"隐藏 upside"

### 3. 仓位管理的期权性

- 杠铃策略（[[antifragility]]）：90% 稳健 + 10% 高 beta
- 关键：右侧必须真凸（损失上限明确）

### 4. 个人决策的期权性

- 副业 / 学习新技能 / 写公开内容（[[naval-four-leverages]]）
- 下行：浪费一些时间
- 上行：改变职业 / 收入 / 生活
- 这是普通人最被低估的期权性来源

## 反例：负期权性的陷阱

很多投资策略**反向**——损失无限，收益有限：

### 1. 卖期权（写期权）

写裸期权 = 收 premium（有限）+ 承担无限损失。

很多对冲基金被这个杀死。2018 年 XIV ETN（做空 VIX）一天蒸发 95%。

### 2. 加杠杆

[[1998-ltcm-collapse]] 30 倍杠杆 = 凹回报：
- 上行：放大有限收益
- 下行：放大无限损失

### 3. 卖出"远 OTM"看跌期权

收一点 premium + 承担市场崩盘风险。

**所有"长期稳定收益"的策略**都隐含负期权性——直到崩盘那天才暴露。

## 一个具体数学：仓位为什么要凸

假设你能选择两种押注：
- A：10% 概率赢 10x，90% 概率亏 50%（期望值 = 1.0 - 0.45 = 0.55 → 期望负）
- B：90% 概率赢 1%，10% 概率亏 90%（期望值 = 0.009 - 0.09 = -0.081 → 期望负）

两者期望值都是负的——但**A 是凸的，B 是凹的**。

**Kelly 公式说**：A 应该小仓位试探（潜在大赢），B 应该完全避免（小赢大亏的负偏分布）。

这就是**期权性 + Kelly 公式的组合**——不只看期望值，还看 payoff 形状。

## 反面 / 边界（5 个陷阱）

### 1. 期权 premium 不是免费的

买期权要付 premium。**长期持有期权 = 缓慢出血**。
Universa 99% 时间亏钱就是这个。

需要心理素质 + 资金能扛。

### 2. 隐含波动率高时期权太贵

VIX 80 时（2020 3 月）买保护期权 = 已经太晚。

期权性策略需要**在平静时建仓**——但那时大多数人没动力。

### 3. "Optionality" 被滥用

很多失败创业者说"我在创造期权性"——但他们的下行其实是无限的（时间、信誉、关系）。

真期权性要求**下行真有限**。

### 4. 时间也是成本

期权性需要时间 frame。1 年期权 vs 10 年期权完全不同。**长期是期权性的真朋友**。

### 5. 不是所有"凸"看起来的都凸

很多看似"小赌注"实际隐含巨大下行。
- 加密小币：下行不只是金钱，还有信誉
- 风投：下行包括时间 + 关系
- 创业：下行可能是 10 年青春

**真正的凸 payoff 需要 honest accounting of downside**。

## 给投资者的启示（5 条）

1. **追求凸 payoff，避免凹 payoff**——这是 Taleb 的核心忠告，可以应用到任何决策。

2. **多个小赌注 > 少数大赌注**——Bezos 模式。组合层面才看得到期权性。

3. **承认期权性策略大部分时间表现差**——99% 时间小亏，1% 时间巨赚。心理素质决定能否执行。

4. **个人决策也要期权性思维**：副业、技能、关系、内容输出——下行有限上行无限的事情值得做。

5. **看公司分析时主动找"隐性期权"**——AWS 1999、Apple Services 2016、NVDA CUDA 2010s。这些**当时不显著但后来改变公司命运**的小项目就是期权性来源。
