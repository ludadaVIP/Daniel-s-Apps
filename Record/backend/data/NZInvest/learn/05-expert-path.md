# 从 0 到专家

## 一句话

投资专家不是知道更多 ticker 的人，而是能稳定完成三件事的人：**看懂资产、看懂自己、看懂规则**。

## 0-3 个月：生存与地图

目标：避免大错。

学习内容：

- 术语地图：平台/产品/税务/策略。
- NZ 税务：PIE、PIR、FIF、W-8BEN、遗产税。
- 资产配置：现金、股票、债券、房产、外汇。
- 决策日志：每次买卖前写原因。

合格标准：

- 能解释为什么“直接买 VTI”和“买 NZ PIE 全球基金”不是同一件事。
- 能写一页 IPS。
- 能列出给税务师的关键问题。

## 3-12 个月：建立核心仓

目标：让主要资金有规则地工作。

学习内容：

- 宽基指数：全球、美国、新西兰、澳洲。
- 基金费用：管理费、平台费、买卖价差、外汇点差。
- 再平衡：按比例而不是按情绪。
- 风险承受：历史回撤、熊市周期、通胀。

合格标准：

- 能说清组合里每个产品的角色。
- 能做一年一次的再平衡计划。
- 不因为新闻改变核心策略。

## 1-2 年：读懂公司和基金

目标：从“买指数”进阶到“理解资产质量”。

学习内容：

- 三张报表。
- 自由现金流。
- ROIC、ROE、ROA。
- 护城河、定价权、网络效应。
- DCF、owner earnings、反向 DCF。

练习：

- 每月读一家公司的年报。
- 每次只研究一个问题：赚钱能力、资本效率、竞争优势、估值。
- 写一页研究笔记，不写长篇流水账。

## 2-3 年：宏观与行为金融

目标：知道什么时候是自己错，什么时候只是市场波动。

学习内容：

- 利率、通胀、汇率、债务周期。
- 股债相关性。
- 市场情绪与估值周期。
- 损失厌恶、确认偏差、锚定、社会认同、FOMO。

练习：

- 每季度写一次组合复盘。
- 把错误分成三类：分析错、行为错、运气差。
- 用 pre-mortem 预先写失败原因。

## 长期：形成系统

系统包括：

- IPS：每年更新。
- 关注清单：只收录有 thesis 的标的。
- 决策日志：买入、卖出、加仓、减仓都记录。
- 税务文件夹：报表、成本、汇率、PIR、W-8BEN、税务师意见。
- 复盘节奏：季度看行为，年度看结构。
